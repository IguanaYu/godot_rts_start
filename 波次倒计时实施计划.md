## Context
当前游戏（关卡3、4）有波次敌人系统，但玩家没有任何UI提示——不知道下一波什么时候来。需要在屏幕顶部显示倒计时。

## 2. 修改 `scripts/main.gd`

### 2a. 新增成员变量（第72行 `gold_label` 之后）
```gdscript
var wave_countdown_label: Label
```

### 2b. 在 `_create_ui()` 末尾（第300行 `canvas.add_child(gold_label)` 之后）添加倒计时 Label
- 锚点：水平居中，顶部偏移55px（在建筑按钮栏下方）
- 字号20，红色 `Color(1, 0.3, 0.3)`
- 默认隐藏
- 格式：`"Wave X/Y incoming in: Zs"`
- 最后5秒变亮红色 `Color(1, 0.1, 0.1)`

### 2c. 修改 `_setup_wave_manager()`（第154-160行）
替换现有 lambda 为具名函数，并连接 `countdown_updated` 和 `all_waves_completed`：
```gdscript
func _setup_wave_manager() -> void:
    for child in get_children():
        if child is WaveManager:
            child.set_game_controller(self)
            child.wave_started.connect(_on_wave_started)
            child.countdown_updated.connect(_on_countdown_updated)
            child.all_waves_completed.connect(_on_all_waves_completed)
            child.start_waves()
            break
```

### 2d. 新增信号处理函数（第161行之后）

**`_on_wave_started(wave_number)`**：重置 `_wave_clear_notified = false`

**`_on_countdown_updated(wave_number, remaining, total)`**：
```gdscript
func _on_countdown_updated(wave_number: int, remaining: float, total: int) -> void:
    if wave_countdown_label == null: return
    var display_wave := wave_number + 1
    var secs := ceili(remaining)
    wave_countdown_label.text = "Wave %d/%d incoming in: %ds" % [display_wave, total, secs]
    wave_countdown_label.visible = true
    if remaining <= 5.0:
        wave_countdown_label.add_theme_color_override("font_color", Color(1, 0.1, 0.1))
    else:
        wave_countdown_label.add_theme_color_override("font_color", Color(1, 0.3, 0.3))
```

**`_on_all_waves_completed()`**：
```gdscript
func _on_all_waves_completed() -> void:
    if wave_countdown_label:
        wave_countdown_label.visible = false
```

## 边界情况

| 场景 | 处理方式 |
|------|---------|
| 关卡1/2没有WaveManager | 遍历无匹配，不连接信号，不创建Label |
| `clear_then_next=true`（关卡3） | 清除敌人后才开始下一波倒计时 |
| `clear_then_next=false`（关卡4） | 连续倒计时 |

## 验证方法

1. **运行关卡3**：3波，应看到倒计时从30s开始，清除敌人后下一波倒计时
2. **运行关卡4**：5波连续，倒计时连续切换
3. **运行关卡1**：无WaveManager，无倒计时UI，无报错
