# 多语言系统实现计划

## Context
游戏目前所有文字都硬编码为英文，没有任何本地化系统。需要添加中/英/日三语言支持，语言切换放在选关界面。

---

## 方案概述

使用 Godot 4 内置的 CSV 翻译系统：
1. 创建 CSV 翻译文件（Godot 自动转为 Translation 资源）
2. 代码中用 `tr("KEY")` 替换所有硬编码文字
3. 选关界面添加语言切换按钮
4. 用 `ConfigFile` 持久化用户选择的语言

---

大约 40+ 条翻译。

### 步骤 2：注册翻译文件到 project.godot
**文件**: `project.godot`

在 `[internationalization]` 段落下添加：
```ini
[internationalization]
locale/translations=PackedStringArray("res://locales/translations.csv")
```

### 步骤 3：修改 level_select.gd
**文件**: `scripts/level_select.gd`

改动点：
1. **替换所有硬编码文字**为 `tr()` 调用：
   - `"Select Level"` → `tr("SELECT_LEVEL")`
   - `"Press Esc to quit"` → `tr("PRESS_ESC_QUIT")`
   - levels 数组中的 `name` 和 `desc` → 用 `tr()` 包裹

2. **添加语言切换按钮**（在标题下方或底部）：
   - 三个按钮：English / 中文 / 日本語
   - 点击后调用 `TranslationServer.set_locale("en"/"zh"/"ja")`
   - 切换后调用一个 `_refresh_ui()` 函数，更新所有已显示的文字
   - 用 `ConfigFile` 保存/加载语言偏好（存到 `user://settings.cfg`）

3. **_ready 时加载保存的语言偏好**，设置 locale

### 步骤 4：修改 main.gd
**文件**: `scripts/main.gd`

改动点（所有硬编码文字替换为 tr()）：

| 位置 | 原文字 | 替换为 |
|------|--------|--------|
| L133 | `"Victory!"` | `tr("VICTORY")` |
| L133 | `"Defeat!"` | `tr("DEFEAT")` |
| L249 | `mode_names` 字典 | 用 `tr()` 调用替换各名称 |
| L289 | `"Gold: %d" % gold` | `tr("GOLD") % gold` |
| L345 | `"PAUSED"` | `tr("PAUSED")` |
| L351-354 | 暂停菜单按钮文字 | `tr("RESUME")` 等 |
| L365 | `"Press ESC to resume"` | `tr("PRESS_ESC_RESUME")` |
| L789-792 | 单位放置提示 | `tr("CLICK_PLACE") % tr(unit_key)` |
| L833-843 | 建筑放置提示 | `tr("PLACE_COST") % [tr(type_key), cost]` |
| L854,857,881,884 | 胜利/失败文字 | `tr("VICTORY")` / `tr("DEFEAT")` |

`mode_names` 字典改为用 tr() key 映射：
```gdscript
var mode_names := {
    PlaceMode.WALL: tr("WALL"),
    PlaceMode.TOWER: tr("TOWER"),
    PlaceMode.SOLDIER: tr("SOLDIER"),
    ...
}
```

注意：游戏中一般不切语言（切语言在选关界面），所以 main.gd 只需要在 _ready 时用 tr() 即可，不需要监听语言变化。

### 步骤 5：修改场景文件中的 Attack Move 文字
**文件**: `scenes/main.tscn` 及 4 个地图场景中的 `"Attack Move (Click)"` 文字

在 `main.gd` 的 `_ready()` 中用代码覆盖：
```gdscript
attack_move_indicator.text = tr("ATTACK_MOVE")
```

---

## 验证方式
1. 运行游戏，选关界面看到三个语言按钮
2. 点击「中文」，界面文字全部变成中文
3. 点击「日本語」，界面文字全部变成日文
4. 点击 English，恢复英文
5. 进入任意关卡，兵种按钮名称、金币显示、暂停菜单都是对应语言
6. 退出重进，语言选择被保留
