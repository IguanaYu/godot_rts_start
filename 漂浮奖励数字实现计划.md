## 步骤 1：新建 `scripts/floating_text.gd`

自包含脚本，参考 [click_effect.gd](scripts/click_effect.gd) 的 tween + queue_free 模式。

```
extends Node2D

var _label: Label

func _ready() -> void:
    _label = Label.new()
    _label.horizontal_alignment = HORIZONTAL_ALIGNMENT_CENTER
    _label.add_theme_font_size_override("font_size", 24)
    add_child(_label)

func setup(text: String, color: Color, world_pos: Vector2) -> void:
    global_position = world_pos
    z_index = 20                        # 高于占领点(5)和点击特效(10)

    _label.text = text
    _label.position = Vector2(-80, -30) # 居中偏移
    _label.size = Vector2(160, 30)
    _label.add_theme_color_override("font_color", color)
    _label.add_theme_color_override("font_shadow_color", Color(0, 0, 0, 0.7))
    _label.add_theme_constant_override("shadow_offset_x", 2)
    _label.add_theme_constant_override("shadow_offset_y", 2)

    scale = Vector2(0.3, 0.3)

    var tween := create_tween()
    tween.set_parallel(true)
    # 0-0.2s：弹出放大
    tween.tween_property(self, "scale", Vector2(1.3, 1.3), 0.2)\
        .set_ease(Tween.EASE_OUT).set_trans(Tween.TRANS_BACK)
    # 0-1.0s：向上漂浮
    tween.tween_property(self, "position:y", position.y - 60, 1.0)\
        .set_ease(Tween.EASE_OUT)
    # 0.2s后：缩回 + 淡出
    tween.chain().set_parallel(true)
    tween.tween_property(self, "scale", Vector2(1.0, 1.0), 0.3)
    tween.tween_property(_label, "modulate:a", 0.0, 0.6)
    # 结束清理
    tween.chain().tween_callback(queue_free)
```

## 步骤 2：修改 `scripts/main.gd`

在 [main.gd:1232](scripts/main.gd#L1232) `add_gold()` 之后添加：

```gdscript
func show_floating_text(text: String, color: Color, world_pos: Vector2) -> void:
    var ft := Node2D.new()
    ft.set_script(load("res://scripts/floating_text.gd"))
    add_child(ft)
    ft.setup(text, color, world_pos)
```

**注意**：`add_child` 必须在 `setup` 之前，因为 `add_child` 触发 `_ready()`（创建 `_label`），之后 `setup` 才能安全使用 `_label`。

## 关键设计决策

- **世界坐标 Node2D**（非 CanvasLayer）：文字跟随 Camera 移动，不遮挡暂停菜单
- **z_index = 20**：高于占领点环(5)和点击特效(10)
- **字体 24px + 黑色阴影**：在各种背景上可读
- **动画 ~1.1s**：弹出(0.2s) → 漂浮(1s) → 缩回+淡出(0.6s)
- **挂载到 Main 节点**（非 CapturePoint）：避免占领点 deactivate 后影响漂浮文字

## 验证方法

1. 启动游戏，将单位移动到占领点
2. 占领完成后观察：金币占领点显示黄色 "+500" 漂浮并淡出，单位占领点显示绿色 "+2 Soldiers" 等
3. 确认文字在建筑和单位之上可见
4. 确认动画结束后节点被自动清理
