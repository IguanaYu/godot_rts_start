## 二、回归测试（正式关卡）

| 待测试 | 验证内容 |
|--------|----------|
| 任务目标 | 目标菱形是否正常闪烁（脉动呼吸替代了原来的方波闪烁） |
| 求救 ping | 黄点脉动是否正常（效果应更平滑） |
| 占领点 | 占领中颜色呼吸是否正常 |
| 波次刷怪 | 刷怪点是否正常显示金色三角（修复了之前的静默失效） |
| 建筑/单位 | 实际游戏中建筑方块是否明显大于单位圆点 |
| 敌人团块 | 大量敌人聚集时是否显示为红色团块而非离散点 |
| 左键跳转 | 点击小地图跳转相机是否正常 |
| 木纹背景 | 小地图九宫格背景是否正确渲染 |
