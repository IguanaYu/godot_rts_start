## 关卡3：暴风前线

**类型**：防守 | **难度**：★★ | **时长**：~12分钟

### 玩法概述
玩家基地面临2条路口的敌人波次进攻。地图右侧有3个敌方据点（2近1远），玩家可以选择出击占领。**每占领一个据点，最终波次的敌人数量会减少**。据点占得越多，最终波次越轻松，但出击本身也有风险。

### 地图
- 尺寸：2000 × 1400
- 地形：沙漠
- 玩家基地在左侧，2条路口（上路和下路）通向右侧
- 3个敌方据点分布在右侧区域

```
  ┌──────────────────────────────────┐
  │  路口A(上)───── 据点1(近)        │
  │                    据点2(近)      │
  │  ┌───┐                           │
  │  │城堡│   路口B(下)──── 据点3(远) │
  │  └───┘                           │
  └──────────────────────────────────┘
```

### 玩家配置
- 金币：3000
- 可建造：墙、箭塔、兵营、士兵、弓箭手
- 可用技能：无

**初始部队**：士兵×3、弓箭手×2
**初始建筑**：城堡 + 兵营

### 敌方据点
| 据点 | 距离 | 守军 | 占领奖励 |
|------|------|------|----------|
| 据点1 | 近（上路口尽头） | 士兵×3 + 弓箭手×2 | 300金币 + 2个士兵 |
| 据点2 | 近（下路口尽头） | 士兵×3 + 弓箭手×2 | 300金币 + 2个弓箭手 |
| 据点3 | 远（深处） | 士兵×5 + 弓箭手×3 + 长矛兵×1 | 500金币 + 2士兵1弓箭手 |

### 敌方波次（4波，从右侧进攻）

| 波次 | 时间 | 敌人 |
|------|------|------|
| 1 | 45秒 | 士兵×5 + 弓箭手×2 |
| 2 | 120秒 | 士兵×7 + 弓箭手×4 |
| 3 | 200秒 | 士兵×8 + 弓箭手×5 + 长矛兵×2 |
| 4（最终波） | 280秒 | 士兵×12 + 弓箭手×6 + 长矛兵×3 |

### 据点效果
每占领1个据点 → 最终波次减少：士兵-3、弓箭手-1

| 占领数 | 最终波次敌人 |
|--------|-------------|
| 0个（不占） | 12兵 + 6弓 + 3矛（最难） |
| 1个 | 9兵 + 5弓 + 3矛 |
| 2个 | 6兵 + 4弓 + 3矛 |
| 3个（全占） | 3兵 + 3弓 + 3矛（最易） |

### 胜利/失败
- **胜利**：存活全部4波 + 消灭所有敌人
- **失败**：城堡被摧毁

### 副目标
- 击杀100个敌人

### 星级评分
| 星级 | 时间 | 死亡 |
|------|------|------|
| ★★★ | 8分钟内 | ≤3 |
| ★★ | 10分钟内 | ≤5 |

## 关卡7：古墓防御

**类型**：防守 | **难度**：★★★ | **时长**：~12分钟

### 玩法概述
经典防守关卡，抵御4波敌人。特色：地图上有2个古墓据点，占领据点是**可选的额外挑战**——占领会给玩家奖励（金币+僧侣），但会导致后续波次的敌人**增强**。玩家自行权衡风险和收益。

**首次引入**：修道院（产僧侣）、僧侣（治疗单位）。

### 地图
- 尺寸：2000 × 1600
- 地形：深色调
- 玩家基地在左下，波次从右侧进攻
- 2个古墓据点在地图中上方

```
  ┌──────────────────────────────────┐
  │    古墓据点A     古墓据点B        │
  │       ◇             ◇            │
  │                     敌人生成→     │
  │  ┌───┐                          │
  │  │城堡│     敌人生成→             │
  │  └───┘                          │
  └──────────────────────────────────┘
```

### 玩家配置
- 金币：4000
- 可建造：墙、箭塔、兵营、修道院、士兵、弓箭手、僧侣
- 可用技能：无

**初始部队**：士兵×3、弓箭手×2、僧侣×1
**初始建筑**：城堡 + 兵营 + 修道院

### 古墓据点
| 据点 | 位置 | 守军 | 占领奖励 | 代价（后续波次增强） |
|------|------|------|----------|-------------------|
| 古墓A | 中上偏左 | 士兵×4 + 弓箭手×2 | 400金币 + 1僧侣 | 波次3和4各+3兵+2弓 |
| 古墓B | 中上偏右 | 士兵×5 + 弓箭手×3 | 500金币 + 2士兵 + 1僧侣 | 波次3和4各+3兵+2弓 |

### 敌方波次（4波，从右侧进攻）

| 波次 | 时间 | 基础敌人 | 占领1个古墓后 | 占领2个后 |
|------|------|---------|-------------|----------|
| 1 | 50秒 | 士兵×5 + 弓箭手×3 | 不变 | 不变 |
| 2 | 130秒 | 士兵×7 + 弓箭手×4 | 不变 | 不变 |
| 3 | 210秒 | 士兵×8 + 弓箭手×5 + 矛×2 | +3兵+2弓 | +6兵+4弓 |
| 4 | 290秒 | 士兵×10 + 弓箭手×6 + 矛×3 | +3兵+2弓 | +6兵+4弓 |

### 胜利/失败
- **胜利**：存活全部4波 + 消灭所有敌人
- **失败**：城堡被摧毁

### 副目标
- 占领全部古墓据点（额外的自我挑战）

### 星级评分
| 星级 | 时间 |
|------|------|
| ★★★ | 8分钟内 |
| ★★ | 10分钟内 |

## 关卡10：限时清除

**类型**：进攻 | **难度**：★★★ | **时长**：~10分钟

### 玩法概述
限时进攻关卡。初始倒计时6分钟，需要在时间内摧毁3个敌方据点。每摧毁一个据点，**倒计时延长90秒**。需要高效规划路线，边打边赚时间。

### 地图
- 尺寸：2000 × 1400
- 地形：草地
- 玩家基地在左下，3个据点分散在右侧

```
  ┌──────────────────────────────────┐
  │     据点A(右上)                   │
  │                 据点B(右中)        │
  │  ┌───┐                           │
  │  │城堡│         据点C(右下)        │
  │  └───┘                           │
  └──────────────────────────────────┘
```

### 玩家配置
- 金币：6000
- 可建造：全部建筑 + 全部兵种
- 可用技能：治疗场、空投部队

**初始部队**：士兵×4、弓箭手×3、长矛兵×2、僧侣×1
**初始建筑**：城堡 + 兵营 + 射箭场

### 3个据点
| 据点 | 位置 | 建筑 | 守军 |
|------|------|------|------|
| A | 右上 | 兵营 + 箭塔 + 墙 | 士兵×5 + 弓箭手×3 |
| B | 右中 | 兵营 + 箭塔 + 墙 | 士兵×6 + 弓箭手×4 + 长矛兵×1 |
| C | 右下 | 兵营 + 2座箭塔 | 士兵×7 + 弓箭手×4 + 长矛兵×2 |

### 倒计时机制
- 初始时间：360秒（6分钟）
- 摧毁每个据点的全部建筑 → +90秒
- 最多可赚取：360 + 90×3 = 630秒（10.5分钟）

### 胜利/失败
- **胜利**：摧毁全部3个据点 + 在倒计时归零前完成
- **失败**：倒计时归零 或 城堡被摧毁

### 副目标
- 4分钟内完成（极速通关）

### 星级评分
| 星级 | 时间 |
|------|------|
| ★★★ | 4分钟内 |
| ★★ | 5分钟内 |

## 关卡15：据点攻防

**类型**：进攻 | **难度**：★★★★ | **时长**：~15分钟

### 玩法概述
攻守转换关卡。需要依次占领3个据点，每占领一个后必须**防守60秒**。敌人会派部队反击试图夺回据点。如果据点被敌人夺回，该据点需要重新攻占。每个据点占领后给予不同资源奖励。

### 地图
- 尺寸：2400 × 1800
- 地形：默认
- 玩家基地在左下，3个据点从近到远排列

```
  ┌──────────────────────────────────────┐
  │                           据点3(远)   │
  │                据点2(中)              │
  │      据点1(近)                        │
  │  ┌───┐                               │
  │  │城堡│                               │
  │  └───┘                               │
  └──────────────────────────────────────┘
```

### 玩家配置
- 金币：7000
- 可建造：全部建筑 + 全部兵种
- 可用技能：全部4个指挥官技能

**初始部队**：士兵×5、弓箭手×3、长矛兵×2、僧侣×2
**初始建筑**：城堡 + 兵营 + 射箭场 + 修道院

### 3个据点
| 据点 | 距离 | 守军 | 占领奖励 |
|------|------|------|----------|
| 据点1 | 近 | 士兵×4 + 弓箭手×2 | 300金币 |
| 据点2 | 中 | 士兵×6 + 弓箭手×3 + 长矛兵×1 | 400金币 |
| 据点3 | 远 | 士兵×8 + 弓箭手×4 + 长矛兵×2 | 500金币 |

### 反击波次（占领据点后触发）
敌人从地图右侧派兵反击该据点：

| 据点被占后 | 反击部队 |
|-----------|---------|
| 据点1 | 士兵×5 + 弓箭手×2 |
| 据点2 | 士兵×7 + 弓箭手×4 + 长矛兵×1 |
| 据点3 | 士兵×10 + 弓箭手×5 + 长矛兵×2 |

### 游戏流程
1. 攻占据点1 → 60秒倒计时防守 → 成功后进入下一阶段
2. 攻占据点2 → 60秒倒计时防守
3. 攻占据点3 → 60秒倒计时防守 → 全部成功即胜利

**据点丢失**：如果在60秒防守期间，敌人把据点夺回，该阶段重置，需要重新攻占。

### 胜利/失败
- **胜利**：3个据点全部占领并各自守住60秒
- **失败**：玩家城堡被摧毁

### 副目标
- 守点期间玩家城堡不受伤

### 星级评分
| 星级 | 时间 | 条件 |
|------|------|------|
| ★★★ | 10分钟内 | 无据点丢失 |
| ★★ | 13分钟内 | — |
