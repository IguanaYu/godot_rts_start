# Phase 1 需求：战役基础设施

## 一、战役解锁系统

### 玩家体验
- 游戏从"4关战役"扩展为"16关战役"
- 关卡按编号顺序**线性解锁**：第1关始终可玩，通关第N关后解锁第N+1关
- 未解锁的关卡在界面上显示为灰色/锁定状态，玩家可以看到但无法进入

### 存档规则
- 现有的3个存档位不变
- 每个存档位独立记录16关的通关进度
- 已有的旧存档自动兼容：原本通关的关卡保持已通关状态，新增关卡显示为未解锁
- 每关记录：是否通关、最快通关时间、游玩次数、得分、星级

## 二、关卡选择界面

### 界面布局
- 左侧面板：可滚动的关卡列表，16个关卡按钮纵向排列
- 右侧面板：选中关卡的预览信息

### 关卡按钮
每个按钮显示：
- 关卡图标（4种图标循环使用）
- 关卡名称（本地化文本）

### 关卡状态

**已锁定（未解锁）**：
- 按钮显示灰色
- 点击无反应，无法选中
- 右侧面板显示"需要通关前一关"或类似提示

**已解锁（未通关）**：
- 按钮显示正常蓝色
- 可以选中并启动
- 右侧面板显示：关卡名称、描述、游玩次数

**已通关**：
- 按钮显示红色（选中态）
- 右侧面板显示：关卡名称、描述、通关状态、得分、最快时间、游玩次数、星级

### 右侧预览面板
- 关卡标题
- 关卡描述文字
- 关卡图标 + 装饰丝带
- 通关信息：得分、最快时间、游玩次数
- 难度选择器（简单/普通/困难）
- 开始按钮
