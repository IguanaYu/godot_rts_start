# Phase 4 需求：新机制关卡

> 逐个新增游戏功能，每加一个功能紧跟对应的关卡。
> 需要新增的5个功能：
> 1. **伏击触发器** — 敌人初始隐藏，玩家走近时突然激活
> 2. **建筑驻军** — 建筑被摧毁时释放内部存储的敌方单位
> 3. **精英光环** — 精英敌人给周围友军加增益效果
> 4. **自适应增兵** — 根据玩家存活单位数动态调整援军频率
> 5. **Boss机制** — Boss无敌护盾 + 定时召唤小兵

---

## 4.1 新功能：伏击触发器 + 关卡4：神射三杰

### 新功能说明：伏击触发器
- 地图上设置一个隐形的检测区域
- 区域内预先放置的敌人初始处于**隐藏+沉睡**状态（不可见、不活动）
- 当玩家单位走进检测区域时，敌人被**激活**——突然出现并开始攻击
- 每个触发器只触发一次

---

### 关卡4：神射三杰

**类型**：特别行动 | **难度**：★★ | **时长**：~8分钟

#### 玩法概述
操控3个特殊精英弓箭手（高攻击、极低血量）+ 4个士兵护卫，穿越一条森林走廊。沿途有3处伏击点，敌人会突然出现。**任何弓箭手死亡即失败**。需要善用士兵挡在前面保护弓箭手，微操走位躲伏击。

#### 地图
- 尺寸：2400 × 600（长条走廊）
- 地形：森林
- 左侧出发，右侧是终点

```
  ┌──────────────────────────────────────────────────────┐
  │起点  伏击A   拐角  伏击B    开阔地   伏击C   终点★   │
  │ →    ⚡        ⚡              ⚡               ★    │
  └──────────────────────────────────────────────────────┘
  0    400      800    1000    1400    1800    2200  2400
```

#### 玩家配置
- 金币：0
- 可建造：无
- 可用技能：无

**初始部队**：
| 单位 | 数量 | 说明 |
|------|------|------|
| 精英弓箭手 | 3 | VIP，必须全部存活。攻击极高（25），血量极低（30） |
| 士兵 | 4 | 护卫。标准属性 |

#### 伏击配置
| 伏击点 | 位置 | 触发范围 | 敌人 |
|--------|------|---------|------|
| 伏击A | x≈500 | 较小 | 士兵×3 + 弓箭手×2 |
| 伏击B | x≈1100（拐角处） | 中等 | 士兵×4 + 弓箭手×3 |
| 伏击C | x≈1900（终点前） | 较大 | 士兵×5 + 弓箭手×4 + 长矛兵×1 |

#### 胜利/失败
- **胜利**：3个精英弓箭手全部到达终点
- **失败**：任一精英弓箭手死亡

#### 副目标
- 4个护卫士兵也不死

#### 星级评分
| 星级 | 时间 | 条件 |
|------|------|------|
| ★★★ | 5分钟内 | 全员无损失 |
| ★★ | 7分钟内 | 死亡≤2 |

## 4.2 新功能：建筑驻军 + 关卡6、8

### 新功能说明：建筑驻军
- 某些建筑内部驻扎着敌方单位
- 建筑正常运作时这些单位不会出现
- 当建筑被玩家摧毁时，驻扎的单位会**在建筑废墟附近生成**并开始战斗
- 驻军数量和类型可以在建筑上预先配置

---

### 关卡6：资源枯竭

**类型**：进攻 | **难度**：★★★ | **时长**：~12分钟

#### 玩法概述
金币极少（1500），地图上5个敌方点位（2小+2中+1大）。摧毁点位获得部队增援和金币，**以战养战，滚雪球**。关键是选对进攻顺序——先打弱的补充实力，再打强的。摧毁某些点位时，建筑内的驻军会冲出来，需要提前做好准备。

#### 地图
- 尺寸：2000 × 1600
- 地形：草地
- 玩家在左侧出发，5个点位散布全图

```
  ┌──────────────────────────────────┐
  │  点位1(小)      点位2(中)        │
  │                                   │
  │  起点          点位3(小)          │
  │  ★                    点位4(中)   │
  │                         点位5(大) │
  └──────────────────────────────────┘
```

#### 玩家配置
- 金币：1500（极少，省着花）
- 可建造：仅士兵和弓箭手（无建筑）
- 可用技能：空投部队

**初始部队**：士兵×4、弓箭手×3

#### 5个敌方点位

**点位1(小)** — 左上：

**点位2(中)** — 右上：

- 占领奖励：300金币 + 2个弓箭手

**点位4(中)** — 右中：

**点位5(大)** — 右下：

- 地形：沙漠
- 双方各占一侧，中间开阔地带是战场

```
  ┌──────────────────────────────────────┐
```

#### 玩家配置

- 可建造：全部建筑 + 全部兵种（不含修道院）
- 可用技能：护盾墙

## 4.3 新功能：精英光环 + 关卡9：四方围攻

### 新功能说明：精英光环
- 某些精英敌人自带光环效果
- 光环范围内的所有敌方友军获得增益（如攻击力+8）
- 光环范围约200像素（约2.5个单位距离）
- 精英单位外观更大，方便识别
- 击杀精英单位后光环消失

---

### 关卡9：四方围攻

**类型**：防守 | **难度**：★★★ | **时长**：~12分钟

#### 玩法概述
玩家基地在中央，敌人从东南西北4个方向进攻。每方向的兵种不同（北=近战、东=远程、南=长矛兵、西=混合）。关键机制：**精英敌人**在波次间出现，它们有光环会给周围敌人加攻击力。玩家必须**主动出击**打掉精英，否则防线会被高伤害的强化敌人冲破。

**首次引入**：射箭场（产弓箭手）。

#### 地图
- 尺寸：2400 × 2400（大正方形）
- 地形：默认
- 玩家基地在正中央，4个方向的生成点

```
          北(近战)↑
  ┌──────────────────────┐
  │       ↑生成点          │
  │                       │
  │ ←西    ┌───┐    东→   │
  │ (混合)  │城堡│  (远程)  │
  │         └───┘          │
  │       ↓生成点          │
  │       南(长矛)         │
  └──────────────────────┘
```

#### 玩家配置
- 金币：8000
- 可建造：墙、箭塔、兵营、射箭场、士兵、弓箭手、长矛兵
- 可用技能：治疗场、护盾墙

**初始部队**：士兵×4、弓箭手×3、长矛兵×2
**初始建筑**：城堡 + 兵营 + 射箭场 + 2座箭塔

#### 波次配置（5波，四方向交替）

**波次1**（50秒）— 北+南夹击：
- 北：士兵×6
- 南：长矛兵×4

**波次2**（100秒）— 东+西夹击：
- 东：弓箭手×6
- 西：士兵×4 + 弓箭手×2

**波次3**（160秒）— 四方向+1个精英：
- 北：士兵×8
- 南：长矛兵×5
- 东：弓箭手×5 + 士兵×3
- 西：士兵×5 + 弓箭手×3
- **精英**（从北面出现）：体形巨大(×1.8)，自带攻击光环(范围内友军攻击+8)，HP=250

**波次4**（230秒）— 四方向+2个精英：
- 四方向各有更多敌人
- **精英A**：攻击光环(范围内友军攻击+8)
- **精英B**：远程型精英(弓箭手变体)，攻击光环(范围内友军攻击+6)

**波次5**（300秒）— 最终大波+3个精英：
- 四方向各：士兵×8 + 弓箭手×4 + 长矛兵×2
- **3个精英单位**分别从北、东、西出现

#### 胜利/失败
- **胜利**：存活全部5波 + 消灭所有敌人
- **失败**：城堡被摧毁

#### 副目标
- 玩家建筑不被摧毁

## 4.4 新功能：自适应增兵 + 关卡11：源源不绝

### 新功能说明：自适应增兵
- 场景上挂载一个定时增兵系统
- 每隔一段时间自动为玩家生成援军
- **核心机制：玩家部队越少，援军来得越快越多**
- 玩家部队多时，援军少且慢；部队被打残时，大量援军快速补充
- 保证玩家始终有战斗能力，不会陷入"死局"

---

### 关卡11：源源不绝

**类型**：特别行动 | **难度**：★★★ | **时长**：~12分钟

#### 玩法概述
纯爽局。玩家没有基地，只有初始部队。面对一个大型敌方多层防御基地，需要一层一层打穿。**核心机制：部队越少，来的援军越多越快**。不用担心全军覆没，大胆进攻就好。无硬失败条件。

#### 地图
- 尺寸：2000 × 1600
- 地形：草地
- 玩家在左侧出发，大型敌方基地在右侧，3层防线

```
  ┌──────────────────────────────────────┐
  │  起点     外围防线    内圈    核心城堡  │
  │  ★→  墙+塔+兵   墙+塔   ┌───┐       │
  │              墙+塔+兵     │城堡│       │
  │                          └───┘       │
  └──────────────────────────────────────┘
  0    400     800    1200   1600  2000
```

#### 玩家配置
- 金币：0
- 可建造：无
- 可用技能：无

**初始部队**：士兵×8、弓箭手×4、僧侣×2

#### 自适应增兵规则
| 存活单位数 | 援军间隔 | 每次给的援军 |
|-----------|---------|-------------|
| 少于5 | 8秒 | 3兵 + 2弓 |
| 5~10 | 15秒 | 2兵 + 1弓 |
| 10~15 | 20秒 | 1兵 + 1弓 |
| 超过15 | 25秒 | 1兵 |

援军出现在玩家起始位置附近。

#### 敌方基地（3层防线）

**外围防线**（第1层）：
- 3面墙 + 3座箭塔，均匀分布
- 守军：士兵×6 + 弓箭手×4

**内圈防线**（第2层）：
- 2面墙 + 2座箭塔 + 1座兵营
- 守军：士兵×8 + 弓箭手×5 + 长矛兵×2
- 兵营驻军：士兵×4 + 弓箭手×2

**核心城堡**（第3层）：
- 1座城堡 + 4座箭塔（四角环绕）
- 守军：士兵×10 + 弓箭手×6 + 长矛兵×3

#### 胜利/失败
- **胜利**：摧毁敌方城堡
- **失败**：无硬失败（援军不断到来，总能打过）

#### 副目标
- 击杀150个敌人

#### 星级评分

|------|------|
| ★★★ | 8分钟内 |

## 4.5 新功能：Boss机制 + 关卡16：终极决战

### 新功能说明
**Boss无敌护盾**：
- Boss单位初始处于无敌状态（受到的伤害减少95%）
- 场景上有若干"护盾发生器"建筑
- 只有全部护盾发生器被摧毁，Boss才会失去无敌状态
- 护盾解除时会有视觉反馈

**Boss召唤小兵**：
- Boss每隔一段时间在自身周围召唤小兵
- 召唤的小兵为普通敌方单位
- Boss被击杀后不再召唤

---

### 关卡16：终极决战

**类型**：特色 | **难度**：★★★★☆ | **时长**：~15分钟

#### 玩法概述
最终Boss战。分为3个阶段：
1. **打护盾**：Boss周围有3个护盾发生器（各有守军），Boss处于几乎无敌状态。需要分兵去摧毁发生器。
2. **Boss觉醒**：3个发生器全毁后，Boss失去护盾，开始主动进攻玩家基地。
3. **最终击杀**：Boss每20秒召唤一波小兵，玩家需要边防守边集火Boss。

需要综合运用全部4个指挥官技能。

#### 地图
- 尺寸：2400 × 1800
- 地形：深色调
- 玩家基地在左下，Boss和护盾发生器在右上区域

```
  ┌──────────────────────────────────────────┐
  │               护盾A(塔)    护盾B(塔)      │
  │                                    👑Boss │
  │               护盾C(塔)                  │
  │                                           │
  │  ┌───┐                                   │
  │  │城堡│                                   │
  │  └───┘                                   │
  └──────────────────────────────────────────┘
```

#### 玩家配置
- 金币：8000
- 可建造：全部建筑 + 全部兵种
- 可用技能：全部4个指挥官技能（轨道轰炸、治疗场、护盾墙、空投）

**初始部队**：士兵×5、弓箭手×4、长矛兵×3、僧侣×2
**初始建筑**：城堡 + 兵营 + 射箭场 + 修道院 + 2座箭塔

#### 敌方配置

**3个护盾发生器**（各有守军保护）：
| 发生器 | 守军 |
|--------|------|
| 护盾A | 士兵×5 + 弓箭手×3 + 长矛兵×1 |
| 护盾B | 士兵×5 + 弓箭手×3 + 长矛兵×1 |
| 护盾C | 士兵×6 + 弓箭手×4 + 长矛兵×2 |

**Boss**：
- 生命值：500（极高，普通士兵只有100）

- 体形：巨大（2.5倍普通单位）
- 移动速度：较慢

#### 胜利/失败
