> 完成后累计可玩：6关（含已有1-2关）
> 以下4关仅使用现有游戏系统配置，无需开发任何新功能。

## 关卡5：技能觉醒

**类型**：特别行动 | **难度**：★★ | **时长**：~10分钟

### 玩法概述
玩家拥有一个小基地和少量初始部队，面对3波敌人进攻。这关的核心是教学两个指挥官技能：**轨道轰炸**（对指定区域造成大量伤害）和**护盾墙**（临时生成阻挡墙）。

### 地图
- 尺寸：1600 × 1200
- 地形：草地
- 玩家基地在左侧中央，敌人生成区在右侧
- 布局简单，一条主战场通道

```
  ┌──────────────────────────────────┐
  │          波次1（右中路）           │
  │  ┌───┐                           │
  │  │城堡│  ←玩家基地                │
  │  └───┘            ┌──┐           │
  │        ┌兵营┐      │墙│ 敌人生成区│
  │        └───┘      └──┘           │
  │          波次3（大规模右侧）        │
  └──────────────────────────────────┘
```

### 玩家配置
- 金币：2000
- 可建造：墙、箭塔、士兵、弓箭手
- 可用技能：轨道轰炸、护盾墙

**初始部队**：
| 单位 | 数量 | 说明 |
|------|------|------|
| 士兵 | 3 | 前排肉盾 |
| 弓箭手 | 2 | 后排输出 |

**初始建筑**：城堡 + 兵营

### 敌方波次

**波次1**（60秒后到来）：
- 4个士兵，从右侧冲锋
- 设计意图：敌人排成一线前进，正好适合轨道轰炸（范围伤害）一波清掉

**波次2**（波次1清除后30秒）：
- 5个士兵 + 3个弓箭手，从右上方进攻
- 设计意图：8个敌人较多，需要用护盾墙挡住冲锋保护己方

**波次3**（波次2清除后25秒）：
- 8个士兵 + 5个弓箭手 + 2个长矛兵
- 设计意图：15个敌人，必须同时用好两个技能才能撑过

### 胜利/失败
- **胜利**：存活全部3波 + 消灭所有敌人
- **失败**：城堡被摧毁

### 副目标
- 无损通关（不损失任何单位）

### 星级评分
| 星级 | 时间 | 死亡 |
|------|------|------|
| ★★★ | 6分钟内 | ≤1 |
| ★★ | 9分钟内 | ≤3 |
| ★ | 通关即可 | — |

## 关卡12：步步推进

**类型**：进攻 | **难度**：★★★☆ | **时长**：~15分钟

### 玩法概述
经典RTS进攻关卡。玩家从左侧出发，沿走廊依次攻克3个敌方据点，逐步向右推进。每摧毁一个据点可以把战线推前，建立前线基地。最终目标是摧毁最右侧的敌方大本营。

### 地图
- 尺寸：3000 × 1000（长条形走廊）
- 地形：默认
- 玩家基地在左侧，3个敌方据点从左到右排列

```
  ┌──────────────────────────────────────────────────┐
  │  玩家    │  据点1   │   据点2    │   据点3       │
  │  基地    │  敌方营地 │   敌方营地  │   敌方大营地   │
  │          │  +箭塔    │   +兵营    │   +城堡       │
  └──────────────────────────────────────────────────┘
  0       800      1200     1600    2000    2400   3000
```

### 玩家配置
- 金币：5000
- 可建造：全部建筑 + 全部兵种
- 可用技能：轨道轰炸、空投部队

**初始部队**：士兵×4、弓箭手×3、长矛兵×2
**初始建筑**：城堡 + 兵营

### 敌方配置

**据点1 — 前哨站**（距起点约800）：
- 建筑：箭塔 + 2面墙
- 守军：士兵×3、弓箭手×2

**据点2 — 中继营地**（距起点约1700）：
- 建筑：兵营 + 箭塔 + 墙
- 守军：士兵×5、弓箭手×3、长矛兵×1

**据点3 — 敌方大本营**（距起点约2500）：
- 建筑：城堡 + 兵营 + 2座箭塔
- 守军：士兵×8、弓箭手×5、长矛兵×2

### 胜利/失败
- **胜利**：依次摧毁3个据点的全部建筑（分3阶段，逐个击破）
- **失败**：玩家城堡被摧毁

### 副目标
- 僧侣不死（如果配置了僧侣的话）

### 星级评分
| 星级 | 时间 | 死亡 |
|------|------|------|
| ★★★ | 10分钟内 | ≤3 |
| ★★ | 13分钟内 | ≤6 |

## 关卡13：截杀密使

**类型**：防守 | **难度**：★★★☆ | **时长**：~12分钟

### 玩法概述
玩家需要防守基地的同时，派出部队击杀3个敌方VIP密使。密使从地图边缘出现，缓慢向敌方基地移动，带有护卫部队。如果密使成功到达敌方基地，后续敌人会增强。需要在防守和出击之间做好平衡。

### 地图
- 尺寸：2000 × 1600
- 地形：沙漠
- 玩家基地在中间偏左，敌方基地在右上角

```
  ┌──────────────────────────────┐
  │                    ┌──┐ 敌基地│
  │         VIP3 →    │  │→→→→ │
  │    VIP2 →         └──┘      │
  │                  ↑           │
  │      ┌───┐    VIP1路径       │
  │      │城堡│                  │
  │      └───┘                  │
  │  VIP起点                      │
  └──────────────────────────────┘
```

### 玩家配置
- 金币：5000
- 可建造：全部建筑 + 全部兵种
- 可用技能：全部4个指挥官技能

**初始部队**：士兵×4、弓箭手×3、长矛兵×2
**初始建筑**：城堡 + 兵营 + 射箭场

### VIP密使

| 密使 | 出现时间 | 起点 | 移动方向 | 护卫 | 生命值 |
|------|---------|------|----------|------|--------|
| VIP1 | 120秒 | 左侧边缘 | 向右上方敌基地 | 士兵×3+弓箭手×2 | 高(300) |
| VIP2 | 300秒 | 下方边缘 | 向右上方敌基地 | 士兵×4+弓箭手×2+长矛兵×1 | 很高(400) |
| VIP3 | 480秒 | 左下边缘 | 向右上方敌基地 | 士兵×5+弓箭手×3+长矛兵×2 | 极高(500) |

密使移动缓慢，给玩家追杀的时间窗口。

### 敌方进攻波次（从右侧攻击玩家基地，制造分兵压力）
- 波次1（60秒）：士兵×6 + 弓箭手×3
- 波次2（240秒）：士兵×8 + 弓箭手×4 + 长矛兵×2
- 波次3（420秒）：士兵×10 + 弓箭手×6 + 长矛兵×3

### 胜利/失败
- **胜利**：击杀全部3个VIP密使 + 存活全部防守波次
- **失败**：玩家城堡被摧毁

### 副目标
- 无一漏网（所有密使到达敌基地前全部击杀）

### 星级评分
| 星级 | 时间 | 条件 |
|------|------|------|
| ★★★ | 8分钟内 | 全部VIP击杀无一到达 |
| ★★ | 10分钟内 | 死亡≤5 |

## 关卡14：圣殿守护

**类型**：特色 | **难度**：★★★☆ | **时长**：~12分钟

### 玩法概述
纯治疗微操关卡。玩家操控4个僧侣 + 2个士兵，保护3个精英单位（攻击力极高但血量极低）。5波敌人从四面八方进攻，僧侣必须持续治疗精英单位防止它们阵亡。**允许多个僧侣同时治疗同一个目标**。

### 地图
- 尺寸：1200 × 1000（较小，集中战斗）
- 地形：深色调/暗色
- 开阔区域，精英单位在中央，僧侣环绕

```
  ┌────────────────────────┐
  │      敌人生成区(N)       │
  │  敌←    ☆精英1    →敌   │
  │  敌  ☆精英2  ☆精英3  敌  │
  │  僧侣x4 + 士兵x2 环绕    │
  │      敌人生成区(S)       │
  └────────────────────────┘
```

### 玩家配置
- 金币：0（无经济）
- 可建造：无（无建筑可建）
- 可用技能：无

**初始部队**：
| 单位 | 数量 | 说明 |
|------|------|------|
| 僧侣 | 4 | 治疗者，是主要操作对象 |
| 士兵 | 2 | 肉盾，帮忙挡伤害 |
| 精英单位 | 3 | 高攻低血，必须保活 |

**精英单位属性**：
- 攻击力：极高（普通士兵的3倍）
- 生命值：极低（只有普通士兵的30%，约30HP）
- 体形略大，方便识别

### 敌方波次（5波，从不同方向进攻）

| 波次 | 间隔 | 敌人 | 来向 |
|------|------|------|------|
| 1 | 30秒后 | 士兵×4 | 北面 |
| 2 | 清后20秒 | 士兵×5 + 弓箭手×2 | 西面 |
| 3 | 清后15秒 | 士兵×6 + 弓箭手×4 | 南面 |
| 4 | 清后15秒 | 士兵×8 + 弓箭手×5 + 长矛兵×2 | 东面 |
| 5 | 清后10秒 | 士兵×10 + 弓箭手×6 + 长矛兵×3 | 北面 |

### 胜利/失败
- **胜利**：存活全部5波 + 3个精英单位全部存活
- **失败**：任一精英单位死亡 或 全军覆没

### 副目标
- 僧侣和士兵也不死

### 星级评分
| 星级 | 时间 | 条件 |
|------|------|------|
| ★★★ | 8分钟内 | 精英HP始终>50% |
| ★★ | 10分钟内 | 非精英死亡≤2 |
