# 关卡战役计划 — 16关线性解锁

## Context

当前游戏有4张地图、灵活的胜负条件系统、波次管理器、据点系统。需要扩展为16关的完整战役。参考星际2战役/合作、魔兽3战役的设计原则：**类型交替、渐进复杂度、每关教1-2个新机制**。

## 二、逐关详细设计

### 第1关：初次出击（特别行动）
- **描述**：5兵+3弓，沿单线走廊推进，清除3个敌方小营地，摧毁终点基地
- **胜利**：VictoryDestroyBase（摧毁敌方城堡）
- **失败**：全灭
- **配置**：`initial_gold=0, available_items=[], player_units=[5兵+3弓]`
- **教学重点**：移动、攻击、攻击移动(A键)、基础编队
- **新代码**：无
- **副目标**：BonusObjectiveNoLosses（无损通关）

### 第2关：直捣黄龙（进攻）
- **描述**：经典RTS。玩家有城堡+兵营，建造经济和部队，消灭地图对面的敌方基地。敌人会定期派小波攻击。
- **胜利**：VictoryDestroyBase
- **失败**：城堡被毁
- **配置**：`initial_gold=5000, available_items=[墙,塔,城堡,兵营,士兵,弓箭手]`
- **教学重点**：建造建筑、产兵、金币管理、防守反击
- **新代码**：无
- **副目标**：BonusObjectiveSpeedRun（8分钟内完成）

### 第3关：暴风前线（防守）
- **描述**：玩家基地面临2个靠近路口的敌人波次进攻。地图上有3个敌方据点（2近1远），玩家可选择外出占领。**占领导致最后一波敌人减少**。胜利条件是坚守并消灭最后一波敌人。
- **胜利**：VictorySurviveWaves（存活全部波次+消灭残敌）
- **失败**：城堡被毁
- **配置**：`initial_gold=3000, available_items=[墙,塔,兵营,士兵,弓箭手]`
- **教学重点**：据点系统（可选出击）、防守节奏、双线防御
- **新代码**：极少（据点captured信号回调减少最终波次单位数，~10行）
- **副目标**：BonusObjectiveKillCount（击杀100敌人）

### 第4关：神射三杰（特别行动）
- **描述**：操控3个特殊精英弓箭手（高攻低血）+4个士兵护卫。穿越森林区域，沿途有敌方**伏击**。任何弓箭手死亡即失败。全部到达终点即胜利。
- **伏击定义**：敌方单位初始处于静止/隐匿状态（在树林或拐角后），当玩家单位进入触发区域（Area2D）时激活，切换为攻击状态向玩家冲锋。本质上是用区域触发的敌人波次，制造"突然被包围"的体验。
- **胜利**：VictoryReachLocation(require_all=true, required_units=[3英雄弓])
- **失败**：任一英雄弓箭手死亡
- **配置**：`initial_gold=0, available_items=[], player_units=[3精英弓+4兵]`
- **教学重点**：微操保护VIP、阵型站位、应对伏击
- **新代码**：伏击触发器（Area2D检测+激活敌人AI，~15行），或用WaveManager的延迟触发替代
- **副目标**：BonusObjectiveNoLosses（士兵也不死）

### 第5关：技能觉醒（特别行动·技能教学）
- **描述**：玩家有一个小基地和初始部队。面对3波敌人进攻。第1波教学**轨道轰炸**（点击地面位置，对范围内敌人造成大量伤害）。第2波教学**护盾墙**（范围内友军短时间减伤）。第3波需要综合使用两个技能才能撑过。
- **胜利**：VictorySurviveWaves
- **失败**：城堡被毁 或 全灭
- **配置**：`initial_gold=2000, available_items=[墙,塔,士兵,弓箭手], commander_skills=[ORBITAL_STRIKE, SHIELD_WALL]`
- **教学重点**：轨道轰炸的AOE时机、护盾墙的防御时机、能量管理
- **新代码**：无（指挥官技能系统已有，WaveManager配3波即可）
- **副目标**：BonusObjectiveNoLosses（不损失单位——需要用好护盾墙）

### 第6关：资源枯竭（进攻）
- **描述**：金币很少(1500)。地图上5个敌人点位：**2小+2中+1大**。
  - 小型据点：3-4个敌人单位，无箭塔。奖励2-3单位+200-300金币
  - 中型据点：8-10个敌人+箭塔。奖励3-4单位+300-400金币
  - 大型基地：大量单位+多箭塔。奖励大量单位+大量金币
  - 摧毁任一据点获得该据点的部队增援+金币，连锁击破滚雪球。
- **胜利**：VictoryComposite(AND) = 摧毁全部5个据点
- **失败**：全灭
- **配置**：`initial_gold=1500, available_items=[士兵,弓箭手], commander_skills=[UNIT_DROP]`
- **教学重点**：目标选择策略、以战养战、滚雪球
- **前置功能**：建筑存兵系统（中型/大型据点的部分敌人存在建筑中，建筑被摧毁时释放）
- **新代码**：建筑存兵释放机制（建筑被摧毁时生成内部存储的单位）
- **副目标**：BonusObjectiveCollectAll（收集据点旁散落的物品）

### 第7关：古墓防御（防守）
- **描述**：防守基地抵御敌人波次。地图上有古墓据点（可选占领）。**占领据点是额外的挑战**——给玩家更多奖励（金币/单位），但会导致敌人波次增强。玩家自行决定是否给自己加难度。
- **胜利**：VictorySurviveWaves（守住并消灭最后一波敌人）
- **失败**：城堡被毁
- **配置**：`initial_gold=4000, available_items=[墙,塔,兵营,修道院,士兵,弓箭手,僧侣]`
- **教学重点**：僧侣治疗、风险/收益评估、首次引入修道院
- **新代码**：极少（据点captured信号增加后续波次难度，~10行）
- **副目标**：占领全部古墓据点（额外的挑战目标）

### 第8关：拉锯战（进攻）
- **描述**：双方都有城堡经济。敌方基地不断"囤兵"（建筑里存兵），定期派囤积的兵力进攻。玩家必须主动出击消耗敌方储备，否则被压垮。引入长矛兵用于快速突袭。
- **胜利**：VictoryDestroyBase
- **失败**：城堡被毁
- **配置**：`initial_gold=6000, available_items=[全建筑全兵种-修道院], commander_skills=[SHIELD_WALL]`
- **教学重点**：经济vs军事平衡、时机判断、长矛兵快攻
- **前置功能**：建筑存兵系统（同第6关）
- **新代码**：无（复用第6关的建筑存兵机制）
- **副目标**：BonusObjectiveKillCount（击杀80敌人）

### 第9关：四方围攻（防守）
- **描述**：玩家基地在中央。敌人从东南西北4个方向进攻，每方向兵种不同（北=近战、东=远程、南=长矛兵、西=混合）。关键机制：**特殊精英敌人**会在波次间出现，它们有特殊能力（如回血光环、增伤光环），会大幅强化周围敌人。玩家必须**主动出击**打掉这些精英，否则防线会崩溃。
- **胜利**：VictorySurviveWaves
- **失败**：城堡被毁
- **配置**：`initial_gold=8000, available_items=[墙,塔,兵营,射箭场,士兵,弓箭手,长矛兵], commander_skills=[HEAL_FIELD, SHIELD_WALL]`
- **教学重点**：多方向防御、出击打高价值目标、射箭场首次出现
- **新代码**：精英敌人单位（UnitStats配特殊光环效果，~20行光环逻辑）
- **副目标**：BonusObjectiveNoLosses（建筑不被摧毁）

### 第10关：限时清除（进攻）
- **描述**：6分钟倒计时内清除3个敌方据点。每个据点摧毁后奖励金币+90秒时间。限时压力下的高效进攻。
- **胜利**：VictoryComposite(AND) = 摧毁3据点 + VictoryTimeLimit(360s)
- **失败**：超时 或 城堡被毁
- **配置**：`initial_gold=6000, available_items=[全建筑全兵种], commander_skills=[HEAL_FIELD, UNIT_DROP]`
- **教学重点**：时间管理、效率、僧侣维持推线不撤退
- **新代码**：极少（据点摧毁回调修改计时器，~10行）
- **副目标**：BonusObjectiveSpeedRun（4分钟内完成）

### 第11关：源源不绝（特别行动·爽局）
- **描述**：玩家定期自动获得援军。**核心机制：部队越少给的越多，部队越多给的越少**。让玩家一直保持战斗节奏，不用担心全军覆没。进攻一个大型敌方多层防御基地。无基地建设，纯爽局。
- **胜利**：VictoryDestroyBase
- **失败**：无硬失败（可以一直获得援军直到通关）
- **配置**：`initial_gold=0, available_items=[], player_units=[8兵+4弓+2僧侣]`
- **教学重点**：持续进攻节奏、爽快感
- **新代码**：AdaptiveReinforcementTimer（检测玩家存活单位数，动态调整援军量，~30行）
- **副目标**：BonusObjectiveKillCount（击杀150敌人）

### 第12关：步步推进（进攻）
- **描述**：沿线性路径推进3个敌方据点。每摧毁一个据点，可在该位置建立前线基地。逐步把战线推前。
- **胜利**：VictoryMultiStage（3阶段，每阶段=摧毁据点+到达下一区域）
- **失败**：城堡被毁
- **配置**：`initial_gold=5000, available_items=[全建筑全兵种], commander_skills=[ORBITAL_STRIKE, UNIT_DROP]`
- **教学重点**：前线基地建设、集结点管理、多基地运营
- **新代码**：无（据点占领后解锁该区域建造）
- **副目标**：BonusObjectiveNoLosses（僧侣不死）

### 第13关：截杀密使（防守）
- **描述**：防守基地的同时，敌方会派出**3个VIP密使单位**（高血量、移动缓慢、持续朝敌方基地移动）。密使带有护卫部队。如果密使到达敌方基地，下一波敌人增强。**击杀全部3个密使即算胜利**（同时也要存活到最后）。
- **胜利**：VictoryComposite(AND) = VictoryAssassinate(击杀3个VIP) + VictorySurviveWaves
- **失败**：城堡被毁
- **配置**：`initial_gold=5000, available_items=[全建筑全兵种], commander_skills=[全部4技能]`
- **教学重点**：分兵多线操作、编队控制、战略价值判断
- **新代码**：极少（VIP密使=高HP+慢速的EscortNPC沿路径走，VictoryAssassinate已有）
- **副目标**：BonusObjectiveNoLosses（在所有密使到达前全部击杀，无一漏网）

### 第14关：圣殿守护（特色·治疗关）
- **描述**：操控4僧侣+2士兵+**3个精英单位**（高攻击力但初始血量很低，需要治疗维持）。面对多波敌人进攻。精英单位血少但火力猛，僧侣必须持续治疗。**允许多个僧侣同时治疗同一个单位**。
- **胜利**：VictoryComposite(AND) = VictorySurviveWaves + 精英单位存活
- **失败**：任一精英单位死亡 或 全灭
- **配置**：`initial_gold=0, available_items=[], player_units=[4僧侣+2兵+3精英(低血)]`
- **教学重点**：纯治疗微操、治疗优先级、多僧侣协作治疗
- **前置功能**：允许多个僧侣同时治疗同一目标（需检查当前治疗逻辑是否冲突）
- **新代码**：检查/修改僧侣治疗机制允许多僧侣同治疗（~10-20行）
- **副目标**：BonusObjectiveNoLosses（僧侣和士兵不死）

### 第15关：据点攻防（进攻）
- **描述**：3个敌方据点需攻占。**每占一个据点后，必须防守60秒反击**。据点丢失需重新攻占。每个据点提供不同战略资源。
- **胜利**：VictoryMultiStage（3阶段，每阶段=占点+守60秒）
- **失败**：城堡被毁
- **配置**：`initial_gold=7000, available_items=[全建筑全兵种], commander_skills=[全部4技能]`
- **教学重点**：攻守转换、快速建立防御、多据点管理
- **新代码**：无（VictorySurviveTimer(60s)做守点，CapturePoint做占点）
- **副目标**：BonusObjectiveNoLosses（守点期间城堡不受伤）

### 第16关：终极决战（特色·Boss战）
- **描述**：最终Boss战。玩家需先摧毁Boss周围的3个护盾发生器（建筑），使Boss失去无敌状态。护盾发生器各有守军。Boss解除无敌后开始主动进攻玩家基地。玩家需要在防守Boss进攻的同时，集火击杀Boss。Boss高HP、高攻击、带特殊技能（周期性召唤小兵）。
- **胜利**：VictoryComposite(AND) = 摧毁3护盾发生器 + VictoryAssassinate(击杀Boss)
- **失败**：城堡被毁
- **配置**：`initial_gold=8000, available_items=[全建筑全兵种], commander_skills=[全部4技能]`
- **教学重点**：综合运用所有学到的机制、分阶段作战、Boss战节奏
- **新代码**：Boss无敌护盾机制（~15行，检测护盾发生器是否全毁）、Boss召唤小兵（~10行定时器）
- **副目标**：BonusObjectiveSpeedRun（12分钟内完成）
