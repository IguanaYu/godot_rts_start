三、星际2 vs 魔兽3 任务标记系统对比

⭐ 星际争霸2（自由之翼/虫群之心/虚空之遗）

  • 3D信标模型悬浮在目标头顶，带向上发射的光柱
  • Beacon_Shield：蓝绿色旋转盾牌 → "必须存活"
  • Beacon_Attack：红色准星十字 → "击杀目标"
  • Beacon_Objective：金色星形 → "到达/收集"
  • Beacon_Defend：类似盾牌 → "防守此区域"
  • Beacon_Explore：问号 → "探索区域"
  • 所有信标始终可见，战争迷雾中半透明
  • 信标跟随目标单位移动，自带旋转动画
  • 小地图上对应闪烁彩色圆点
  • 右侧面板有文字任务列表：绿色=完成 红色=失败
  • 任务变更时有指挥官头像弹窗播报
  • 代表任务："黑暗深处"保护神器、"究极黑暗"坚守

🗡️ 魔兽争霸3（混乱之治/冰封王座）

  • 没有3D信标，主要靠文字和简单2D图标
  • 感叹号 !（金色）：2D面片悬在任务NPC头顶 → 接任务
  • 问号 ?：任务完成可交还
  • 英雄有特殊脚底光圈（比普通单位更大更华丽）
  • 英雄血条更大、有装饰边框
  • 英雄有彩色光环（玩家颜色）
  • 左上角有英雄头像始终可见
  • 没有专门的"必须存活"标记 — 靠任务文字传达
  • F9打开任务日志：金币图标=主线，银币=可选
  • 小地图有黄色 ! 标记任务位置
  • 关键目标通过过场动画+对话传达
  • 任务物品有金色光效

🏰 帝国时代4（战役任务系统）

  • 2D面片图标悬浮在目标头顶（类似本方案）
  • 绿色盾牌：需保护的目标
  • 红色骷髅/准星：需击杀/摧毁的目标
  • 黄色菱形：移动/到达目标点
  • 图标脉冲闪烁吸引注意力
  • 英雄单位有金色名牌 + 装饰血条
  • 小地图上闪烁彩色图标（形状+颜色）
  • 左侧面板始终显示任务列表
  • 主线金色高亮，支线白色
  • 完成任务显示绿色勾号

⚡ 命令与征服系列

  • 泰伯利亚之日/红警2：无世界空间标记，纯靠任务简报文字
  • 将军/绝命时刻：屏幕边缘2D箭头指向目标
  • C&C3 泰伯利亚战争：首次引入脉冲黄色菱形路点
  • 红警3：最完善 — 浮动目标图标（星/盾/准星）+ 小地图彩色图标
  • 侧边栏文字任务列表，完成后绿色划线
  • 任务变更时NPC头像弹窗播报（与SC2类似）

🎖️ 英雄连 / 铁收割

  • 2D面片图标悬浮在目标头顶（盾牌/骷髅/星/旗帜）
  • 屏幕边缘箭头：目标不在视野时出现方向指示（大多数RTS没有）
  • 小地图用图标形状+颜色（旗帜/盾牌/骷髅/星），不仅圆点
  • 所有标记有脉冲动画
  • Iron Harvest沿用CoH设计，开发者来自CoH团队
  • 战役结构采用"WC3模式"：三个阵营组成一个大叙事
  • 头顶信息（overhead info）可缩放或隐藏

六、推荐采用的标记体系（适合你的游戏）

  • 骷髅 Skull（红色 #FF4444）→ 刺杀/击杀目标
  • 准星 Crosshair（橙红 #FF6D00）→ 摧毁建筑

📍 导航类

  • 星形 Star（金色 #FFD740）→ 主要目标位置
  • 旗帜 Flag（蓝色 #42A5F5）→ 护送/占领/到达
  • 菱形 Diamond（黄色 #FFC107）→ 次要路径点
