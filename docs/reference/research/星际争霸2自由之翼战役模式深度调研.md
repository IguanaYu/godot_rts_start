# 星际争霸2：自由之翼 战役模式深度调研

## 一、整体架构：枢纽 + 任务选择

自由之翼的核心设计模式是 **"枢纽(Hub) → 任务(Mission) → 返回枢纽"** 的循环。玩家大部分时间在休伯利安号（Hyperion）上，通过舰桥的星图选择下一个任务，完成后再回到船上。

```
[开场动画/教程关] → [休伯利安号枢纽] → [选择任务] → [战前动画] → [执行任务] → [战后动画] → [回到枢纽]
                                                                                    ↓
                                                                             NPC对话/升级/招募
```

## 二、休伯利安号（枢纽）的4个区域

| 区域 | 功能 | NPC | 设计意义 |
|------|------|-----|---------|
| **舰桥** | 星图选任务、查看已完成任务、重玩关卡、回顾过场动画 | 马特·霍纳（副官） | 主线流程控制中心 |
| **军械库** | 花费资金升级单位（每个单位2选1） | 罗瑞·斯旺 | 有限的升级选择，制造策略深度 |
| **实验室** | 研发科技、神族支线任务 | 斯台特曼 | 收集研发点数解锁科技 |
| **酒吧** | 招募雇佣兵、玩小游戏"失落的维京战机" | 希尔 | 休闲+额外战力投入 |

关键设计点：**每个区域有一个NPC驻守**，玩家可以与NPC对话触发剧情对话，了解背景故事，获得情感代入。

## 三、任务线结构：分支 + 可选 + 主线

战役有 **26个主线任务 + 多个分支任务**，按故事线组织：

### 1. 玛·萨拉章节（开场，线性）
- 自由日 → 不法之徒 → 零点行动（教程性质的3关）

### 2. 休伯利安号到达后（开放选择）
玩家可以自由选择做哪条任务线，每条线有多个任务：

| 任务线 | 触发条件 | 任务数量 | 特点 |
|--------|---------|---------|------|
| **神器线（泰凯斯）** | 主线 | 5关 | 收集萨尔纳加神器碎片 |
| **帝国线（马特）** | 主线 | 6关+1隐藏 | 反抗帝国，收集蒙斯克罪证 |
| **移民线（韩森博士）** | 可选 | 3关 | **分支选择**：拯救 or 毁灭 |
| **幽魂线（托许）** | 可选 | 3关 | **分支选择**：帮幽魂 or 帮诺娃 |
| **预言线（泽拉图）** | 收集水晶后解锁 | 4关 | 神族单位，揭示世界观背景 |
| **查尔线（维勒安）** | 最终章节 | 4关 | **分支选择**：2种进攻路线 |

### 3. 分支选择机制
关键设计：某些任务线在最后有一个**二选一的分支**：
- 韩森线：拯救海文 vs 海文的陷落（只能选一个）
- 托许线：营救托许 vs 幽灵一击（只能选一个）
- 查尔线：野兽之腹 vs 天崩地坼（只能选一个）

选择的关卡会**永久影响剧情走向**，但不会影响是否能通关。这种设计增加了重玩价值。

## 四、升级/经济系统：双重资源

### 1. 军费（金钱）
- 每完成一个任务获得固定奖励
- 用于：军械库升级 + 雇佣兵招募
- 核心设计：**资金有限，无法全部升级** → 玩家必须做出取舍

### 2. 研发点数（特殊资源）
- 神族研发点：战场上收集，用于升级神族科技
- 异虫研发点：战场上收集，用于升级异虫科技
- 多余的点数可以1:10000兑换军费

### 3. 军械库升级机制
每个单位有2个升级选项，只能二选一：
- 机枪兵：+10HP vs 强化护盾
- 医疗兵：治疗加速 vs 能量上限+25
- 等等... 共约15个单位的二选一升级

## 五、战前/战后流程设计

每个任务的完整流程：
```
1. 在舰桥选择任务星球
2. → 战前过场动画（1-2分钟，交代剧情背景）
3. → 任务简报（任务目标、奖励）
4. → 加载地图，进入战斗
5. → 任务完成，触发战后动画
6. → 回到休伯利安号
7. → 与新NPC对话触发新剧情
8. → 去不同区域升级/招募
```

**关键设计**：战后回到枢纽时，NPC会有新的对话内容，玩家可以去不同区域"串门"，触发剧情进展。有些NPC只在特定任务后才出现。

## 六、游戏内剧情表现手法

战斗中剧情通过以下方式呈现：
- **任务中语音对话**：单位/英雄在特定时刻触发对话
- **任务中途过场**：完成某个子目标后触发短动画
- **目标更新文字**：剧情推进时更新任务描述
- **NPC头像弹出**：雷诺或其他角色在UI上弹出头像对话
- **环境叙事**：地图上的场景、残骸、中立单位讲述故事

## 七、对RTS游戏的启发总结

| 自由之翼的设计 | 可以借鉴的点 |
|--------------|-------------|
| 休伯利安号枢纽 | 做一个"基地/营地/地图"界面，选关卡、升级、看剧情 |
| NPC驻守各区域 | 每个系统绑定一个角色，增强剧情代入感 |
| 任务线分支 | 关键节点让玩家二选一，增加重玩价值 |
| 有限升级资源 | 不要让玩家能全升满，迫使做策略选择 |
| 战前/战后动画 | 任务前后各一段短剧情（30秒-2分钟） |
| 任务中剧情 | 战斗中触发对话、中途过场 |
| 主线+支线混合 | 有必须通关的主线，也有可选的支线 |
| 经济/科技双资源 | 一种货币升级装备，另一种解锁特殊能力 |
