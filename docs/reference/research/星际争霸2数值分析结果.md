- **护甲收益递减的临界点**：当攻击力≤3 时，2 甲后触发 0.5 下限，再升级收益极低
- **SC2 护甲范围 0-3 是合理的**：超过 3 甲对低伤害单位形成绝对免疫（如 4 甲+对 3 伤害=打不动）

## 9. 造价回归估算

### 9.1 每单位资源购买的 HP×DPS

| 种族 | 所有单位均值 | 战斗单位均值 | 说明 |
|------|------------|------------|------|
| **Terran** | 7.80 | 7.80 | 均衡，MULE 经济优势在面板外 |
| **Protoss** | 8.39 | 8.39 | 略高，护盾提供了额外 eHP 不体现在面板 |
| **Zerg** | 8.55 | 7.87 | 略高但含 Zergling（极低价单位）拉高均值 |

### 9.2 拟合公式

用 HP×DPS 预测造价（排除法系和纯功能单位后）：

```
Cost ≈ HP×DPS / 8.0
```

即：**每 8 点 HP×DPS 价值 1 单位资源（1矿或0.5气）**。

**偏差分析：**
- 面板性价比显著高于族内均值（>20%）：Zealot, Cyclone, Ultralisk → 可能有隐性弱点（近战、科技线长）
- 面板性价比显著低于族内均值（<-20%）：Reaper, Ghost, Phoenix → 隐性价值高（技能、机动、骚扰）

## 10. 综合结论

### 10.1 暴雪数值设计验证

1. **减法公式 + 0.5 下限** — 护甲收益非线性，对高频低伤单位形成天然克制
2. **Tier 梯度** — T1→T3 HP×DPS 约 ×10，造价约 ×7.6，高级兵"贵得有道理"
3. **气体杠杆** — 纯矿兵 DPS/人口 6-20，耗气兵 2-7，气体有效控制强度
4. **种族平衡** — 三族 HP×DPS/Cost 均值 7.8-8.6，差异 <10%
5. **Overkill 作为隐性平衡** — 高伤兵种打小兵有 10-30% 效率损失，鼓励混编

### 10.2 可应用到本项目的数值规则

| 规则 | 具体值 |
|------|--------|
| T1 HP 范围 | 30-50 |
| T2 HP 范围 | 80-180 |
| T3 HP 范围 | 200-500 |
| DPS/人口上限（纯矿兵） | ≤20 |
| DPS/人口范围（耗气兵） | 2-7 |
| 护甲范围 | 0-3 |
| TTK（同级互打） | 2-8s |
| 性价比基准 | Cost ≈ HP×DPS / 8 |
| 气体价值系数 | 气:矿 = 2:1 |
| 经济饱和 | 16采集单位/矿 |

### 10.3 异常单位（值得关注的设计案例）

| 单位 | 异常点 | 设计解释 |
|------|--------|---------|
| Cyclone | 性价比 T2 第 1（34.29） | 锁定技能需要引导，实战 DPS 远低于理论值 |
| Colossus | 性价比极低（面板） | AOE 溅射价值未计入，实际团战价值翻倍 |
| Reaper | 性价比 T1 最低 | 翻越悬崖+KD8 炸弹，骚扰价值占大部分成本 |
| Zergling | DPS/人口 20.00 | 近战脆皮，实战中大部分在走到目标前就死了 |
