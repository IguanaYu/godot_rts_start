## 第一部分：阵营/领主设计分析

### 1. 欧洲领主设计哲学

Stronghold 中的欧洲阵营代表了中世纪封建城堡战争的全部要素：

- **重装步兵为核心**：剑士、长枪兵拥有金属铠甲，生存力极强，适合正面攻坚和城防驻守。
- **弓兵体系完整**：弓兵造价低、射速快；弩手伤害高、穿甲强，但需要皮革甲 + 弩的生产链。两者配合可形成弹幕。
- **骑士是终极兵种**：需要剑 + 金属甲 + 马匹（马厩），是游戏中最强近战单位，兼具高速度和高伤害，适合冲锋和追击。
- **城堡防守导向**：石墙、塔楼、城门、护城河、杀戮坑、沥青沟构成多层防御。弓兵上塔获得射程和视野加成，弩手在塔上几乎无敌。
- **经济复杂但回报高**：欧洲阵营拥有完整的武器锻造链（弓匠铺、车床工棚、铁匠铺、皮匠铺、盔甲铺），需要大量投资但一旦建成可源源不断产出重装单位。
- **缓慢但不可阻挡**：欧洲军队集结速度慢（需要锻造武器 + 训练），但成型的重装军团几乎无法被轻装部队阻挡。

设计意图：欧洲阵营让玩家体验"经营-锻造-集结-出征"的完整城堡领主生活，强调经济深度和战略耐心。

### 2. 十字军领主设计哲学

Stronghold Crusader 引入的阿拉伯阵营代表了沙漠机动作战和骚扰战术：

- **轻装 + 廉价 + 快速**：奴隶兵仅 5 金，投石兵 12 金，阿拉伯弓兵 15 金——全部不需要武器生产链，从雇佣兵哨站直接用金币购买。
- **火攻是核心特色**：奴隶兵可以点燃木质建筑和沥青沟；掷火兵投掷火焰罐造成范围伤害。在沙漠干燥环境下，火攻具有毁灭性效果。
- **高机动骚扰**：骑射手是游戏中最灵活的单位之一，高速移动 + 远程攻击，可以风筝任何欧洲步兵。
- **刺客突破防线**：刺客可以隐身并攀爬城墙，用于暗杀城墙上的弓兵或突袭领主，是攻城战的奇兵。
- **无需经济链**：阿拉伯单位只需要金币，不需要锻造武器。这意味着只要经济（贸易或税收）够强，可以瞬间爆兵。
- **正面战力不足**：阿拉伯阵营缺少重装步兵和重骑兵，在正面决战中处于劣势。需要依靠机动性和火攻来消耗敌人。

设计意图：十字军阵营让玩家体验"快节奏、低成本、高骚扰"的沙漠战术，与欧洲阵营的"慢节奏、高成本、重装甲"形成鲜明对比。两阵营的设计体现了中世纪十字军东征中东西方军事体系的碰撞。

### 3. AI 领主个性分析

Stronghold Crusader 中的 AI 领主各有独特的战略性格，是游戏最具魅力的设计之一：

#### 欧洲阵营 AI

**The Rat（老鼠公爵 / Duc Truffe）**
- 难度：★☆☆（最弱 AI）
- 性格：胆小、无能、爱抱怨。建造简陋的小型方形城堡，薄墙少塔。
- 战术：只用弓兵和矛兵进行小规模骚扰，从不组织大规模进攻。经济薄弱，容易被碾压。
- 玩家对策：早期一波骑兵冲锋即可消灭。

**The Snake（蛇王 / Lord Volpe）**
- 难度：★★☆
- 性格：阴险、狡猾、善用地形。建造迷宫式城堡，多层墙体 + 护城河。
- 战术：大量使用弩手上塔防守，配合掘地道兵破坏敌方攻城器械。喜欢偷袭对方经济建筑。
- 玩家对策：需要攻城器械拆墙，注意保护后方经济。

**The Pig（猪公爵 / Duc Truffe）**
- 难度：★★☆
- 性格：粗鲁、贪婪、爱美食。城堡偏重经济（大量农场），防御中等。
- 战术：偏爱锤兵 + 弩兵组合，频繁发动中等规模突袭。不善于防守，经济虽强但军事效率低。
- 玩家对策：防守住他的突袭后反击，他的城堡防御较弱。

**The Wolf（狼王 / Duc Beauregard）**
- 难度：★★★★★（最强欧洲 AI）
- 性格：冷酷、精明、军事天才。建造巨型多层石堡，厚墙高塔 + 护城河 + 杀戮坑。
- 战术：使用骑士、剑士、长枪兵组成精锐军团，配合投石机攻城。经济极度优化，城堡几乎无法正面攻克。
- 玩家对策：需要大量攻城器械 + 重装部队，准备持久战。

#### 十字军阵营 AI

**Saladin（萨拉丁）**
- 难度：★★★☆
- 性格：高贵、智慧、均衡。建造设计精良的城堡，攻守兼备。
- 战术：全阿拉伯兵种搭配使用，骑射手骚扰 + 奴隶兵纵火 + 刺客突袭。经济管理优秀，会根据战况调整策略。
- 玩家对策：需要防骑射手的塔楼弓兵，注意防火。

**The Caliph（哈里发）**
- 难度：★★★
- 性格：暴躁、好战、喜欢火。城堡较小但布满沥青沟和火陷阱。
- 战术：早期疯狂用奴隶兵纵火 + 掷火兵进攻，试图在游戏前期烧毁玩家经济。后期较弱。
- 玩家对策：前期用弓兵守住，别用木质建筑；后期反推容易。

**Richard Lionheart（狮心王理查）**
- 难度：★★★★
- 性格：勇猛、正派、重装主义。建造坚固城堡，偏重军事。
- 战术：大量生产骑士和剑士，发动重装冲锋。几乎不用轻装单位。防守时弩兵上塔。
- 玩家对策：长枪兵阵克制骑士冲锋，攻城时小心他的骑士出城反冲锋。

**Frederick（腓特烈皇帝）**
- 难度：★★★★
- 性格：帝王气派、重装至上。建造宏大城堡，资源充沛。
- 战术：与狼王类似，使用重装步兵军团。但比狼王更偏进攻，经常主动出击。
- 玩家对策：需要坚固的城防 + 长枪兵反骑，消耗他的精锐部队。

#### 其他 AI 领主（扩展）

**The Marshal（元帅）**
- 难度：★★★
- 性格：老练军人，战术多样。使用混合兵种，善于防守反击。

**The Abbot / Nizar（修道院长 / 尼扎尔）**
- 难度：★★★
- 性格：宗教/暗杀主题。使用刺客和狂信徒，擅长奇袭。

**The Shah（沙赫）**
- 难度：★★★
- 性格：波斯贵族，偏爱骑射手。大量生产骑射手进行游击战，经济一般。

**Phillip（法兰西国王菲利普）**
- 难度：★★★☆
- 性格：骄傲的国王，使用重装部队但不如狼王精明。城堡华丽但防守有漏洞。

### 4. 城堡防御体系

Stronghold 的城堡防御是该系列的核心玩法，分为以下层次：

| 墙体类型 | 造价 | HP | 特点 |
|---|---|---|---|

#### 4.2 塔楼系统

| 塔楼类型 | 造价 | 容量 | 特点 |
|---|---|---|---|
| 瞭望塔 (Lookout Tower) | 15 木/10 石 | 2-3 人 | 最便宜，视野加成，防守力弱 |

### 5. 经济模型

Stronghold 的经济系统是该系列最具深度的设计之一，包含食物链、武器链和资源采集三大体系。

#### 5.1 资源类型

| 资源 | 采集建筑 | 采集方式 | 用途 |
|---|---|---|---|
| 木材 (Wood) | 伐木场 (Woodcutter's Hut) | 伐木工砍树自动运回 | 建筑建造、弓/矛武器、攻城器械 |
| 石料 (Stone) | 采石场 (Quarry) | 石匠采石，牛车运输 | 城墙、塔楼、石门 |
| 铁矿 (Iron Ore) | 铁矿场 (Iron Mine) | 矿工在铁矿点挖掘 | 锤/剑/金属铠甲 |
| 沥青 (Pitch) | 沥青场 (Pitch Rig) | 在沼泽地采集 | 沥青沟、火盆、火焰武器 |
| 牛只 (Cows) | 奶牛场 (Dairy Farm) | 牧场饲养 | 奶酪（食物）+ 皮革（铠甲） |

#### 5.2 食物链

Stronghold 的食物系统有多种类型，**食物多样性直接提升民心（popularity）**：

```
食物链 1：苹果
  果园 (Orchard, 5木) → 苹果 → 粮仓 (Granary)
  最简单的食物链，建造快，产量中等

食物链 2：奶酪
  奶牛场 (Dairy Farm, 5木) → 奶酪 → 粮仓
  同时产出牛只供皮匠铺使用，双重收益

食物链 3：面包（最复杂）
  麦田 (Wheat Farm, 5木) → 小麦 → 磨坊 (Mill, 10木+5石)
  → 面粉 → 面包房 (Bakery, 5木) → 面包 → 粮仓
  产量最高但需要3个建筑，投资大

食物链 4：肉类
  猎人小屋 (Hunter's Hut, 5木) → 猎杀地图上的鹿 → 肉 → 粮仓
  产量受地图鹿群数量限制，不可控
```

**民心机制**：
- 每种食物类型提供 +1 民心（最多 +4，四种食物都有时）
- 配给量：双倍配给 +2 民心，正常 0，半量 -2，无食物 -4
- 民心 > 50：农民持续涌入；民心 < 50：农民离开
- 税收：收税降低民心，免税/发钱提升民心
- 啤酒 (Ale)：啤酒花田 → 酿酒厂 → 啤酒馆 (Inn) → +民心
- 宗教 (Religion)：教堂/大教堂 → +民心

#### 5.3 武器生产链（核心特色）

Stronghold 的武器系统是分层的：原材料 → 武器工坊 → 军械库 → 兵营训练。

```
========== 欧洲武器生产链 ==========

弓 (Bow)：
  木材 → 弓匠铺 (Fletcher's Workshop, 10木) → 弓 → 军械库 (Armory)
  生产速度：中。3木材 → 1弓（约）

弩 (Crossbow)：
  木材 → 弓匠铺 → 弩 → 军械库
  生产速度：慢。比弓消耗更多木材和时间

矛 (Spear)：
  木材 → 车床工棚 (Poleturner's Hut, 10木) → 矛 → 军械库
  生产速度：快。少量木材 → 1矛

长枪 (Pike)：
  木材 → 车床工棚 → 长枪 → 军械库
  生产速度：中。比矛消耗更多木材

锤 (Mace)：
  铁矿 → 铁匠铺 (Blacksmith's Workshop, 20木) → 锤 → 军械库
  生产速度：中。1铁矿 → 1锤

剑 (Sword)：
  铁矿 → 铁匠铺 → 剑 → 军械库
  生产速度：慢。1铁矿 → 1剑（但生产时间长）

皮革甲 (Leather Armor)：
  牛只 → 皮匠铺 (Tanner, 10木) → 皮革甲 → 军械库
  牛只来源：奶牛场产出，自动走向皮匠铺

金属甲 (Metal Armor)：
  铁矿 → 盔甲铺 (Armorer's Workshop, 20木) → 金属甲 → 军械库
  生产速度：慢。1铁矿 → 1金属甲

马匹 (Horse)：
  马厩 (Stable, 20木) → 自动产马
  训练骑士时消耗1匹马

========== 阿拉伯武器链 ==========

阿拉伯单位不需要武器生产链！
所有阿拉伯单位从雇佣兵哨站 (Mercenary Post, 10木) 直接用金币雇佣。
这是阿拉伯阵营的核心设计优势——无需锻造链，爆兵极快。
```

**武器链建筑汇总**：

| 皮匠铺 (Tanner) | 10 木 | 牛只 | 皮革甲 | 牛只来自奶牛场 |
| 盔甲铺 (Armorer's Workshop) | 20 木 | 铁矿 | 金属甲 | 生产最慢的武器 |
| 马厩 (Stable) | 20 木 | — | 马匹 | 自动产马，供骑士用 |
| 军械库 (Armory) | 5 木 | — | (存储) | 所有武器的存储点 |

### 6. 核心协同组合

#### 组合 1：弓兵 + 矛兵城防（经典欧洲防守）

```
配置：城墙/塔楼上的弓兵 × 20-30 + 城门后的矛兵 × 10-15
战术：弓兵从墙头射杀来犯之敌（获射程+高度加成），矛兵守城门防骑兵冲入
优势：成本极低（弓=弓+20金，矛=矛+8金），矛兵对骑兵有加成
克制：攻城器械拆墙 + 弩手对射
```

#### 组合 2：弩手 + 长枪兵方阵（欧洲重防阵线）

```
配置：弩手 × 15-20 + 长枪兵 × 10-15
战术：弩手提供高伤害远程火力（穿甲），长枪兵在前方吸收伤害并反骑兵
优势：弩手一击重甲单位伤害极高，长枪兵 HP 60 + 金属甲极难击杀
劣势：移动速度慢，不适合进攻，仅适合阵地防守
克制：投石机范围伤害 + 骑射手风筝
```
