## 第一部分：纪元设计分析

---

### 1.1 EE1 的纪元演进设计哲学（从原始到未来的史诗跨度）

EE1 的核心特色是 **14 个纪元横跨 50 万年**，从石器时代的原始部落到纳米时代的未来科技。这是 RTS 游戏史上最宏大的时间跨度设计。

**设计哲学核心**：

1. **无缝时间线** — 不像 AoE 系列分多个游戏，EE1 用一个游戏涵盖全部历史。玩家可以在一场对局中从原始部落打到未来机甲。
2. **纪元门槛设计** — 每个纪元需要消耗资源"升级时代"（在 Town Center/Capitol 研究），类似 AoE 的时代升级但更频繁。升级成本逐纪元递增，迫使玩家在"升时代"与"爆兵"之间抉择。
3. **单位线延续性** — 每个兵种有跨纪元的升级链（如弓兵→弩手→火枪→步枪→坦克），让玩家在升时代后旧兵种自动获得升级路径，而非废弃重建。
4. **建筑功能演化** — 同一建筑在不同纪元生产不同单位。Barracks 从棍棒兵一直生产到赛博步兵；Stable 从青铜骑兵生产到工业轻骑兵；Factory 从工业时代接棒生产坦克和机械化单位。
5. **资源体系恒定** — 5 种资源（食物/木材/黄金/石头/铁）贯穿全纪元。早期以食物/木材为主，中后期黄金/铁的重要性上升。铁资源在青铜时代后开始大量使用。
6. **"历史叙事"而非"竞技平衡"** — EE1 的设计更注重历史代入感。一战坦克 vs 中世纪骑士的跨时代对战是有意为之的"史诗感"，而非追求严格对称竞技。

### 1.2 EE2 的文明加成系统（地区 + 文明）

EE2 在 EE1 的基础上引入了 **地区（Region）+ 文明（Civilization）** 两层体系，取代了 EE1 的通用科技树 + 文明加成。

**地区系统（4 大地区）**：

| 地区 | 包含文明 | 特征 |
|---|---|---|
| **Western（西方）** | American, British, French, German, Greek, Roman, Russian | 科技全面，火药/工业优势，完整攻城线 |
| **Middle Eastern（中东）** | Babylonian, Egyptian, Carthaginian, Turk | 贸易/宗教优势，骆驼骑兵，塔防加成 |
| **Far Eastern（远东）** | Chinese, Japanese, Korean | 人口/经济优势，火药早发，大量廉价单位 |
| **Mesoamerican（中美洲）** | Aztec, Inca, Mayan | 农业经济，特殊祭司单位，无骑兵线 |

每个地区共享基础科技树和外观数模，但每个文明有独特的：
- **文明加成**（被动属性加成，如 German +10% 军事伤害）
- **独特单位**（1-2 个，如 Japanese Samurai, British Longbowman）
- **独特科技**（在特定纪元解锁，如 Chinese 的 Gunpowder 技术）

**EE2 纪元变更**：从 EE1 的 14 纪元扩展到 **15 纪元**，后期增加 Genetic Age（基因时代）和 Synthetic Age（合成时代），取代了 EE1 的单一 Nano Age。

### 1.3 EE3 的区域系统（西方/中东/远东三区域）

EE3 大幅简化了 EE1/EE2 的纪元系统，从 14-15 纪元缩减为 **5 个纪元**，同时强化了区域差异化。

**三大区域**：

| 区域 | 设计理念 | 核心机制 | 风格 |
|---|---|---|---|
| **Western（西方）** | 科技与工业强国 | 科技树偏重火药→工业→未来，防御建筑强力，坦克线完整 | 技术碾压，慢热后期强 |
| **Middle Eastern（中东）** | 贸易与宗教网络 | 贸易路线加成，骆驼机动部队，宗教单位可转化敌人，可移动基地 | 经济游击，灵活机动 |
| **Far Eastern（远东）** | 人海与火力压制 | 人口上限高，单位成本低，早期火药（火箭/火炮）， Samurai 线 | 数量压制，进攻型 |

**5 个纪元**：Ancient → Medieval → Colonial → Modern → Future

**EE3 的核心简化**：
- 资源简化为 2-3 种（Wealth + Raw Materials，具体资源名称因版本而异）
- 科技树改为 **辐射状（Radial Tech Tree）**，从中心向外分支，而非线性递进
- 每个区域有完全不同的科技树结构，而非共享基础
- 引入 **World Domination 模式**（回合制策略地图 + RTS 战斗），类似 Total War

### 1.4 核心机制对比

| 机制 | EE1 | EE2 | EE3 |
|---|---|---|---|
| **纪元数** | 14 | 15 | 5 |
| **资源** | 食物/木材/黄金/石头/铁（5种） | 食物/木材/黄金/石头/科技点（5种） | 财富/原材料（2-3种，简化） |
| **士气** | 核心机制：城镇中心/神庙提供士气半径，范围内单位减伤 | 保留但弱化 | 移除 |
| **先知/宗教** | 先知单位可施放灾厄（地震/火山/飓风等） | 保留，成本/效果调整 | 移除，改为区域被动 |
| **英雄** | 两类：战士型（战斗光环）/ 战略家型（治疗+战斗光环） | 保留 | 移除 |
| **皇冠** | 无 | 三皇冠系统：军事/经济/帝国 | 无 |
| **文明差异** | 通用科技树 + 文明加成 | 地区+文明双层体系 | 区域完全差异化 |
| **科技树** | 线性纪元升级 | 线性+皇冠分支 | 辐射状分支 |
| **人口上限** | 默认 200 | 默认 200 | 区域差异化（远东更高） |
| **奇观** | 每个奇观提供独特全局加成 | 保留 | 移除 |

**EE1 士气系统详解**：
- 城镇中心（Town Center）和议会大厦（Capitol）提供士气光环
- 神庙（Temple）也提供士气
- 光环内单位受到的伤害降低（最高约 40% 减伤）
- 士气等级由文化建筑数量决定，更多神庙/城镇中心 = 更高士气
- 攻城武器不受士气影响

**EE2 皇冠系统详解**：
- **军事皇冠（Military Crown）**：通过战斗获取点数，持有后全军攻击+防御加成，可召唤援军
- **经济皇冠（Economic Crown）**：通过经济活动获取点数，持有后采集效率加成，可召唤商队
- **帝国皇冠（Imperial Crown）**：通过领土扩张获取点数，持有后领土加成，可召唤移民
- 每个皇冠同时只能被一个文明持有，可通过积累更多点数夺取

### 1.5 跨纪元单位演进链

EE1 最核心的设计是 **单位线的跨纪元延续**。以下是 6 条主要演进链：

#### 链 1：远程步兵线（弓兵→火枪→步枪→赛博）

```
Bowman (石器/铜器)
  → Composite Bowman (青铜/铁器)
    → Crossbow (中世纪)
      → Arquebus (文艺复兴) — 火药转折点
        → Musketeer (帝国)
          → Fusilier (工业)
            → Marine (二战/现代)
              → Cyborg (纳米)
```

**设计要点**：弓兵线在文艺复兴时代过渡为火枪手，射击方式从箭矢变为子弹，但核心定位（远程步兵）不变。火枪手比弩手射程短但伤害高，体现了火药对弓弩的替代。

#### 链 2：近战步兵线（棍棒→剑→长枪）

```
Clubman (石器)
  → Mace Man (铜器)
    → Short Swordsman (青铜)
      → Phalanx (铁器) — 长矛转型
        → Barbarian (黑暗时代)
          → Long Swordsman (中世纪)
            → Men at Arms (文艺复兴)
```

**设计要点**：近战步兵在中世纪达到顶点（Men at Arms），之后被火枪手取代。Phalanx 是从剑士到长矛兵的过渡——代表铁器时代希腊方阵的崛起。

#### 链 3：反骑兵线（长矛→戟）

```
Hoplite (青铜) — 反骑兵起源
  → Phalanx (铁器)
    → Pikeman (中世纪)
      → Halberdier (文艺复兴/帝国)
```

**设计要点**：独立的反骑兵步兵线，与近战步兵线在 Phalanx 分叉。Hoplite/Phalanx 兼任反骑和正面战斗，Pikeman 以后专职反骑兵。

#### 链 4：骑兵→装甲线（马→坦克→机甲）

```
Cavalry (青铜)
  → Cataphract (铁器)
    → Knight (中世纪)
      → Cuirassier (帝国/文艺复兴)
        → Hussar (工业) — 骑兵线终点
          ↓ (建筑从 Stable 切换到 Factory)
        Tank (一战)
          → Medium Tank (二战)
            → Main Battle Tank (现代)
              → Cyber Tank / Titan (纳米)
```

**设计要点**：骑兵线在工业时代终结，Hussar 是最后的骑兵单位。一战起从 Factory 生产坦克，取代骑兵的冲击角色。这条演进链是 EE1 最具代表性的"从马背到装甲"叙事。

#### 链 5：攻城线（投石→火炮→导弹）

```
Stone Thrower (青铜)
  → Catapult (中世纪) — 升级
  → Trebuchet (中世纪) — 装填/拆卸机制
    → Cannon (文艺复兴/帝国) — 火药转折
      → Artillery (工业)
        → Howitzer (一战/二战)
          → Missile Launcher (现代/数字)
```

#### 链 6：海军线（木船→铁甲舰→航母）

```
War Raft (石器)
  → War Galley (青铜)
    → Frigate (文艺复兴)
      → Galleon (中世纪/帝国)
        → Ironclad (工业)
          → Cruiser (一战)
            → Battleship (二战)
              → Aircraft Carrier (二战/现代)
```

**设计要点**：海军线跨越全纪元，从石器时代的木筏到二战的航空母舰。工业时代是关键转折——风帆战舰被铁甲舰取代。

### 1.6 核心协同组合（5-7 组跨纪元典型组合）

#### 组合 1：石器时代 Rush（石器时代）

| 单位 | 角色 |
|---|---|
| Clubman ×8-10 | 正面冲击 |
| Rock Thrower ×3-5 | 远程输出 |

**战术**：石器时代兵力少，Clubman + Rock Thrower 的组合是最基础的前后排列。Rock Thrower 在后方输出穿刺伤害，Clubman 在前排肉搏。由于此时没有骑兵和攻城武器，步兵对拼就是全部。

---

#### 组合 2：青铜时代合成兵种（青铜时代）

| 单位 | 角色 |
|---|---|
| Hoplite / Short Swordsman | 正面步兵线 |
| Composite Bowman | 远程输出 |
| Cavalry | 侧翼包夹/追击 |
| Stone Thrower | 攻城/反密集阵 |

**战术**：EE1 第一个"完整合成兵种"时代。步兵抗线+弓兵输出+骑兵包夹是标准阵型。Stone Thrower 提供攻城能力和对密集步兵阵的溅射伤害。

---

#### 组合 3：中世纪城堡推进（中世纪）

| 单位 | 角色 |
|---|---|
| Long Swordsman / Men at Arms | 正面步兵 |
| Crossbow | 远程穿刺 |
| Knight | 重骑兵冲锋 |
| Pikeman | 反骑兵保护 |
| Trebuchet | 拆建筑/远程压制 |

**战术**：中世纪是 EE1 最"经典"的冷兵器时代。Trebuchet 拆卸/装填机制让攻城节奏变慢但威力巨大。Knight 是核心突击力量，Pikeman 保护阵型不被对方 Knight 冲穿。

---

#### 组合 4：火药革命（帝国/文艺复兴）

| 单位 | 角色 |
|---|---|
| Musketeer | 远程步兵主力 |
| Halberdier | 反骑兵 |
| Cannon | 攻城/反步兵溅射 |
| Cuirassier | 重骑兵冲锋 |

**战术**：火药时代的关键变化是 Musketeet 取代弓兵成为远程主力——射程短但伤害高。Cannon 提供强大攻城和溅射火力。Cuirassier 是最后的重骑兵，配合 Halberdier 防反骑。

---

#### 组合 5：工业时代线列战争（工业时代）

| 单位 | 角色 |
|---|---|
| Fusilier / Rifleman | 线列步兵 |
| Artillery | 远程炮击 |
| Hussar | 轻骑兵骚扰/追击 |
| Ironclad | 近海炮击支援 |

**战术**：工业时代是冷热兵器交替的尾声。Hussar 是最后的骑兵，用于骚扰和追击。Artillery 提供远程压制。Ironclad 在沿海地图提供强大的对岸炮击。

---

#### 组合 6：二战装甲合成（二战/现代）

| 单位 | 角色 |
|---|---|
| Marine / Infantry | 步兵主力 |
| Tank (Medium) | 装甲突击 |
| Fighter | 制空/反轰炸机 |
| Bomber | 对地溅射/拆建筑 |
| Battleship | 海上火力支援 |

**战术**：EE1 的"现代战争"模板。Tank 是核心突击力量，步兵提供反坦克和占点能力。Fighter 保护 Bomber 执行对地打击。Battleship 在沿海提供压倒性炮火。

---

#### 组合 7：纳米时代未来战（纳米时代）

| 单位 | 角色 |
|---|---|
| Cyborg | 超级步兵 |
| Titan / Cyber Tank | 重型机甲 |
| Hyperion / Advanced Air | 空中优势 |
| Advanced Siege | 远程火力 |

**战术**：纳米时代单位造价高但战力极强。Cyborg 拥有远超 Marine 的火力和生存力。Titan 是地面霸主，类似升级版主战坦克。少量高科技单位的对决取代了数量堆叠。

## 第三部分：单位数据表

### 数据约定

- **资源**: F=食物, W=木材, G=黄金, S=石头, I=铁
- **护甲**: EE1 分为近战护甲(armor)和远程护甲(pierce_armor)，但大部分单位两者相同或为 0
- **攻击类型**: 近战(melee) / 穿刺(pierce) / 攻城(siege) / 枪弹(gun) / 爆炸(explosive)
- **生产建筑**: Barracks(兵营) / Archery Range(靶场) / Stable(马厩) / Siege Workshop(攻城厂) / Dock(码头) / Airport(机场) / Factory(工厂) / Town Center(城镇中心)
- **速度**: 游戏内速度单位（约 1.0-2.5 为地面单位常见范围）
- **冷却**: 秒
- **射程**: 游戏内距离单位
- 数值均为 **近似值**，可能因版本/补丁有 ±10-20% 偏差
