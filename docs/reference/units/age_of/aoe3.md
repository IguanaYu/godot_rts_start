## 游戏概述

帝国时代3决定版包含 **16个可玩文明** (含DLC) + 原住民盟邦系统。游戏核心特色:

### 核心系统

1. **家乡城市卡组系统 (Home City Card System)** — AoE3 最核心特色
   - 每个文明拥有可自定义的卡组(15-25张卡)
   - 通过积累经验值(XP)从家乡城市发送卡片
   - 卡片类型: 单位运送 / 资源运送 / 科技升级 / 特殊能力
   - XP来源: 建造建筑、战斗、采集资源、贸易站

2. **5个时代 (Age)**
   - Discovery Age (发现时代, Age 1): 起始时代, 无军事单位
   - Colonial Age (殖民时代, Age 2): 基础步兵/骑兵
   - Fortress Age (要塞时代, Age 3): 散兵/龙骑兵/轻型炮兵
   - Industrial Age (工业时代, Age 4): 重型炮兵/工厂单位
   - Imperial Age (帝王时代, Age 5): 全部升级/帝国卫队

3. **人口上限**: 默认 200; 房屋提供 10 人口(俄国 5 人口)
4. **克制系统**: 基于单位标签(HeavyInfantry / LightInfantry / HeavyCavalry 等)和伤害倍率
5. **原住民盟邦**: 在原住民聚落建造贸易站 → 解锁原住民战士
6. **贸易路线**: 在贸易路线建造贸易站 → 获得XP或资源

### 16个可玩文明

| 分类 | 文明 |
|---|---|
| 欧洲(原版8) | British, French, Spanish, Portuguese, Dutch, Russian, German, Ottoman |
| 原住民(WarChiefs 3) | Aztec, Lakota (Sioux), Haudenosaunee (Iroquois) |
| 亚洲(Asian Dynasties 3) | Japanese, Chinese, Indians |
| DE新增 | USA, Swedes |
| DLC追加 | Incas, Hausa, Ethiopians, Mexicans (不计入16文明基础) |

## 第一部分: 文明设计分析

### 1. 英国 (British)

**设计哲学**: 通过庄园(Manor)经济爆发实现最强经济, 配合卡组强化打造游戏最强火枪兵线; 家乡城市卡组系统是英国的核心 — 大量火枪兵相关卡片让普通单位变成精英。

**签名单位/卡组支援**:
- **Longbowman (长弓兵)**: 射程 22, 基础步兵最远射程, 持续输出
- **Musketeer (火枪兵)**: 通过卡组加成(Team Musketeer Damage / Musketeer Damage+HP+Speed)成为游戏最强火枪兵
- **Rocket (火箭炮)**: 卡片专属炮兵, 高溅射伤害, 替代重型炮兵
- **Manor House (庄园)**: 建造时自动生成 1 个村民(卡片可提升至 2 个)

**角色分工**:
| 角色 | 单位 | 说明 |
|---|---|---|
| 步兵 | Musketeer + Longbowman | 核心力量 |
| 骑兵 | Hussar / Dragoon | 通用, 无独特单位 |
| 炮兵 | Falconet/Culverin + Rocket(卡片) | 火箭炮为卡片专属 |
| 探险家 | 标准 Explorer | 无特殊能力 |

**核心协同组合**:
1. **Musketeer + Longbowman**: 火枪兵抗线+反骑, 长弓兵后排远射输出 — 克制骑兵+步兵混合编队
2. **Musketeer Rush (Age 2)**: 利用卡片快速运送大量火枪兵, 早期压制
3. **Manor Boom → Age 3 Timing**: 快速经济发展后, 用卡片运送军队打时间差
4. **Rocket + Musketeer (Age 4-5)**: 火箭炮溅射 + 火枪兵保护 — 终极阵地战组合
5. **Redcoat Musketeer (Imperial)**: 帝国级"红衣军"火枪兵, 游戏最强重步兵之一

**强势时间窗**:
- **Age 2**: Musketeer + Longbowman timing
- **Age 3-4**: 经济爆发后的军事压制
- **Age 5**: Imperial Redcoat + Rocket 终极组合

**反制弱点**:
- 早期骑兵薄弱, 依赖火枪兵反骑
- 反炮兵能力一般, 需要龙骑兵或轻骑兵
- 经济爆发需要时间, 怕早期快攻(尤其俄国Strelet Rush)

### 2. 法国 (French)

**设计哲学**: 精英经济+精英军队, 以 Coureur des Bois 和 Cuirassier 为核心, 单兵质量碾压; 卡组系统强化原住民协同, 是"少而精"的典型。

**签名单位/卡组支援**:
- **Coureur des Bois (林地皮毛猎人)**: 240 HP, 比普通村民更强, 采集效率更高, 速度 5.0
- **Cuirassier (胸甲骑兵)**: 550 HP, 近战溅射伤害, 游戏最强重骑兵, 3 人口
- **原住民卡组**: Native Treaties(免费原住民战士) / Native Warrior Combat(原住民加成)
- **Cavalry Combat 卡片**: 骑兵全面加成

**角色分工**:
| 角色 | 单位 | 说明 |
|---|---|---|
| 步兵 | Musketeer / Skirmisher / Crossbowman | 通用, 无独特步兵 |
| 骑兵 | **Cuirassier** + Dragoon | 胸甲骑兵为核心 |
| 炮兵 | 通用炮兵 | 无独特炮兵 |
| 原住民 | 卡组强化 | French-Indian Alliance |
| 探险家 | 标准 Explorer | 无特殊能力 |

**核心协同组合**:
1. **Cuirassier + Skirmisher**: 胸甲骑兵冲锋破阵 + 散兵后排反重步 — 法国经典组合
2. **Cuirassier + Dragoon**: 重骑兵冲锋 + 龙骑兵反骑保护
3. **Native Warrior + Coureur Boom**: 利用原住民卡片快速出兵, 同时经济爆发
4. **Cuirassier Mass (Age 3)**: 3 人口胸甲骑兵海, 压制一切非反骑单位
5. **Imperial Gendarme (Age 5)**: 帝国级胸甲骑兵, 终极碾压形态

**强势时间窗**:
- **Age 2**: Coureur 经济领先(更强村民 = 更快发展)
- **Age 3**: Cuirassier Timing (最强时间窗)
- **Age 5**: Imperial Gendarme 终极形态

**反制弱点**:
- Cuirassier 人口效率低(3 人口), 怕大量反骑单位(Pikeman/Dragoon 海)
- 单兵昂贵, 数量劣势时怕被包围
- 早期军事单位少, 怕快攻

### 3. 西班牙 (Spanish)

**设计哲学**: 快速发货+灵活军事, 通过 Rodelero 和 Lancer 打造最强步兵-骑兵协同; 卡组发货速度最快(需更少XP), 是"灵活适应"的典型。

**签名单位/卡组支援**:
- **Rodelero (盾剑士)**: 速度 5.5, 游戏最快步兵, 专杀骑兵
- **Lancer (长矛骑兵)**: 对步兵有加成的重骑兵, 独特定位(其他文明 Hussar 无步兵加成)
- **War Dog (战犬)**: 探险家可训练, 早期侦查/骚扰
- **Missionary (传教士)**: 教堂独特治疗单位, 骑乘可移动治疗
- **快速发货**: 西班牙需要更少 XP 即可发送卡片

**角色分工**:
| 角色 | 单位 | 说明 |
|---|---|---|
| 步兵 | **Rodelero** + 通用步兵 | 盾剑士为核心反骑 |
| 骑兵 | **Lancer** + Hussar/Dragoon | 长矛骑兵杀步兵 |
| 炮兵 | 通用炮兵 | 无独特炮兵 |
| 海军 | **Galleon** | 可训练单位的运输船 |
| 探险家 | Explorer + **War Dog** | 可训练战犬 |

**核心协同组合**:
1. **Rodelero + Lancer**: 盾剑士反骑 + 长矛骑兵杀步兵 — 完美互补, 西班牙核心组合
2. **Rodelero Rush (Age 2)**: 快速盾剑士压制骑兵文明
3. **Lancer + Crossbowman**: 长矛骑兵 + 弩兵反重步
4. **War Dog + Rodelero (Age 1-2)**: 早期战犬骚扰 + 盾剑士跟进
5. **Missionary + 军队**: 传教士治疗保持军队续航, 配合卡片加成

**强势时间窗**:
- **Age 1-2**: 快速发货优势, 早期军事压制(最强时间窗)
- **Age 3**: Rodelero + Lancer 完整组合
- **Age 4-5**: 卡片强化后的终极步兵-骑兵协同

**反制弱点**:
- 经济加成少, 依赖军事优势滚雪球
- 怕散兵海(Lancer 虽对步兵有加成, 但怕散兵集火)
- 后期经济乏力, 需要前期建立优势

### 4. 俄罗斯 (Russian)

**设计哲学**: 以量取胜, 通过批量训练(Block Training)快速产出大量廉价单位; 卡组强化"人海战术", 用数量淹没对手。

**签名单位/卡组支援**:
- **Strelet (滑膛枪兵)**: 游戏最便宜的散兵, 批量训练(10人一组), 单兵弱但数量碾压
- **Cossack (哥萨克骑兵)**: 1 人口便宜骑兵, 批量训练, 替代 Hussar
- **Oprichnik (特务骑兵)**: 独特袭扰骑兵, 对村民/建筑有加成, 袭击经济线
- **Block Training (批量训练)**: 步兵以约 45% 折扣批量训练(10人一组), 但必须整组训练
- **Fort Wagon (要塞马车)**: 卡片运送, 唯一可建造要塞的欧洲文明

**角色分工**:
| 角色 | 单位 | 说明 |
|---|---|---|
| 步兵 | **Strelet** + Musketeer/Halberdier | 散兵海为核心 |
| 骑兵 | **Cossack** + **Oprichnik** | 哥萨克+特务骑兵袭扰 |
| 炮兵 | 通用炮兵 + Horse Artillery | 移动炮兵 |
| 探险家 | 标准 Explorer | 无特殊能力 |

**核心协同组合**:
1. **Strelet Mass**: 10 人一组快速生产, 散兵海淹没重步兵 — Age 2 最强快攻之一
2. **Strelet + Cossack**: 散兵反重步 + 哥萨克反散兵/袭扰
3. **Oprichnik Raid**: 特务骑兵袭击经济线, 对村民/建筑有加成 — 打乱对手经济
4. **Musketeer + Strelet (Age 3)**: 火枪兵抗线 + 散兵输出
5. **Fort Push**: 卡片运送要塞, 前线压制 — 俄国独有

**强势时间窗**:
- **Age 2**: Strelet Rush (最强时间窗, 游戏最强快攻之一)
- **Age 3**: 批量 Musketeer + Strelet
- **Age 4**: Fort + 大军推进

**反制弱点**:
- 单兵质量极低, 怕溅射伤害(炮兵是天敌)
- 需要数量优势, 被削减后战力骤降
- 经济基础薄弱, 怕长期消耗战
- Strelet 射程短, 怕长弓兵/散兵远程消耗

### 5. 奥斯曼 (Ottoman)

**设计哲学**: 免费村民自动生产+强大炮兵, 以 Janissary 和 Abus Gun 为核心; 卡组系统强化炮兵线, 是"慢热但终局无敌"的典型。

**签名单位/卡组支援**:
- **免费村民**: 城镇中心自动生产村民, 不消耗资源(但无法控制生产节奏)
- **Janissary (耶尼切里)**: 170 HP, 比火枪兵更强, 游戏最强重步兵之一
- **Abus Gun (阿布斯枪)**: 独特轻炮兵, 分类为轻骑兵, 反散兵利器
- **Spahi (斯帕希骑兵)**: 卡片专属重骑兵, 400 HP, 无法常规训练
- **Great Bombard (巨型炮)**: 游戏最强炮兵, 200 HP, 70 伤害, 7 人口
- **Mosque (清真寺)**: 独特建筑, 提供特殊科技升级

**角色分工**:
| 角色 | 单位 | 说明 |
|---|---|---|
| 步兵 | **Janissary** | 耶尼切里为核心 |
| 骑兵 | Spahi(卡片) + Hussar/Dragoon | 斯帕希卡片专属 |
| 炮兵 | **Abus Gun** + **Great Bombard** | 游戏最强炮兵线 |
| 探险家 | 标准 Explorer | 无特殊能力 |

**核心协同组合**:
1. **Janissary + Abus Gun**: 耶尼切里抗线 + 阿布斯枪反散兵 — 奥斯曼经典组合
2. **Janissary Rush (Age 2)**: 强力火枪兵早期压制, 配合免费村民经济
3. **Great Bombard + Janissary (Age 4)**: 巨型炮碾压 + 耶尼切里保护 — 终极阵地战
4. **Spahi + Janissary**: 卡片斯帕希骑兵 + 耶尼切里步骑协同
5. **Free Villager Boom → Military Crush**: 免费村民经济爆发 → 军事碾压

**强势时间窗**:
- **Age 2**: Janissary Rush (强力火枪兵压制)
- **Age 3**: Abus Gun + Janissary (经典组合成型)
- **Age 4**: Great Bombard (最强时间窗, 游戏最强炮兵)
- **Age 5**: Imperial Great Bombard

**反制弱点**:
- 早期无法控制村民生产节奏, 可能导致资源不匹配
- 军事单位昂贵, 需要经济支撑
- 怕反炮兵(Culverin)和快速骑兵突袭炮兵
- 升级慢, 怕快攻(尤其俄国 Strelet Rush)

### 6. 阿兹特克 (Aztec)

**设计哲学**: 无火器无骑兵无炮兵的原住民文明, 以精英步兵+战士祭司(Warrior Priest)为核心; 卡组系统配合火堆(Fire Pit)舞蹈, 实现步兵碾压。War Chief(战争酋长)提供独特光环加成。

**签名单位/卡组支援**:
- **Warrior Priest (战士祭司)**: 治疗并提升周围单位攻击力 10%, 可在火堆跳舞加速生产
- **Jaguar Prowl Knight (美洲豹骑士)**: 隐身, 专杀重步兵, 近战高伤害
- **Eagle Runner Knight (雄鹰骑士)**: 远程反骑兵, 替代龙骑兵
- **Puma Spearman (美洲狮枪兵)**: 反骑兵近战步兵
- **Arrow Knight (弓箭骑士)**: 远程反炮兵, 替代炮兵(射程不如炮兵)
- **Fire Pit (火堆/社区广场)**: 独特建筑, 舞蹈提供各种加成(战斗/治疗/生产速度/经验值)
- **War Chief (战争酋长)**: 独特探险家, 拥有光环加成(提升周围单位HP)

**角色分工**:
| 角色 | 单位 | 说明 |
|---|---|---|
| 步兵 | **全部军事力量** | Jaguar/Eagle/Puma/Macehualtin/Otontin/Coyote |
| 骑兵 | **无** | Coyote Runner 为"伪骑兵"步兵 |
| 炮兵 | **无** | Arrow Knight 替代, Battering Ram 攻城 |
| 原住民 | 本身即原住民文明 | 拥有独特单位线 |
| 探险家 | **War Chief** | 光环加成, 提升周围单位HP |

**核心协同组合**:
1. **Jaguar Prowl Knight + Eagle Runner Knight**: 美洲豹杀重步 + 雄鹰反骑 — 阿兹特克核心组合
2. **Warrior Priest + Infantry Mass**: 战士祭司加成 + 步兵海 — 攻击力+10%和治疗
3. **Puma Spearman + Eagle Runner**: 双重反骑保险, 对抗骑兵文明
4. **Fire Pit Combat Dance + Infantry**: 火堆战斗舞蹈 + 步兵攻击力大幅加成
5. **Macehualtin Rush (Age 2)**: 廉价步兵快攻, 类似俄国 Strelet Rush

**强势时间窗**:
- **Age 2**: Macehualtin Rush (廉价步兵快攻)
- **Age 3**: Warrior Priest + Jaguar Prowl Knight Timing (最强时间窗)
- **Age 4-5**: Eagle Runner + Arrow Knight 反炮兵体系成型

**反制弱点**:
- **无骑兵**, 机动性差, 怕散兵海(无法快速包抄)
- **无炮兵**, 攻城依赖 Arrow Knight 和 Battering Ram, 射程不如炮兵
- 怕大量炮兵(Arrow Knight 射程不如 Falconet/Heavy Cannon)
- 火堆舞蹈需要村民, 影响经济
- 怕远程集火(无骑兵突袭后排)

## 数据说明

1. **数值来源**: 本文数据基于 AoE3 DE 游戏知识编写, 部分数值为近似值, 可能与最新补丁有差异。
2. **基础值**: 所有数值为基础值(未升级), 实际游戏中通过卡片/科技/时代升级会显著变化。
3. **伤害倍率**: AoE3 的伤害倍率系统非常复杂, 部分倍率可能因单位版本(基础/Guard/Imperial)而变化。
4. **抗性**: AoE3 有三种抗性类型 — Ranged(远程) / Melee(近战) / Siege(攻城)。抗性按百分比减少对应类型的伤害。
5. **批量训练**: 俄国独有的 Block Training 机制, 步兵必须 10 人一组训练, 总成本约为单独训练的 45%。
6. **卡片系统**: 卡片是 AoE3 的核心特色, 每个文明的卡组配置极大影响战术风格。建议在游戏中查看具体卡组。
7. **补丁影响**: AoE3 DE 经历多次平衡性补丁, 部分数值可能已调整。建议参考最新补丁说明。
