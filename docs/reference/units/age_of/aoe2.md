### 1. Byzantines（拜占庭）

**设计哲学**：防御反击型文明，以廉价反制单位和建筑血量优势构筑"攻不破的堡垒"。

**签名单位/科技**：
- 独特单位：Cataphract（甲胄骑兵）—— 反步兵重骑兵，Logistica 后获得践踏伤害
- 独特单位：Dromon（希腊火战船）—— 溅射攻击海军
- 独特科技：Greek Fire（城堡，火焰船 +1 射程）、Logistica（帝王，甲胄骑兵践踏 + 反步兵）
- 文明加成：建筑每时代 +10% HP（帝王 +40%）、散兵/长戟兵/骆驼 -25% 造价、火焰船 +20% 攻击、僧侣 +50% 治疗速度、上帝时代 -33% 成本、免费 Town Watch

**角色分工**：

| 角色 | 承担单位 |
|---|---|
| 步兵 | Halberdier（廉价反骑兵）、Cataphract（反步兵骑兵） |
| 弓兵 | Skirmisher（廉价反弓）、Arbalester |
| 骑兵 | Cataphract（核心）、Paladin（无血统） |
| 僧侣 | Monk（+50% 治疗速度） |
| 攻城 | Bombard Cannon、Trebuchet |
| 海军 | Fire Ship（+20% atk）、Dromon、Cataphract 不可上海 |

**核心协同组合**：
1. 廉价 Halberdier + 廉价 Skirmisher = 完美反马 + 反弓防御阵
2. Cataphract + Logistica = 践踏反步兵重骑兵海，碾压步兵文明
3. Fire Ship + Greek Fire = 近海无敌火焰船阵
4. 廉价帝王时代 + 免费 Town Watch = 快速上帝王 + 全图视野
5. 建筑高 HP + Halberdier 防守 = 攻不破的基地

**强势时间窗**：城堡时代（防守反击）、帝王时代（Cataphract 军团 + 廉价垃圾兵海）

**反制弱点**：无经济加成（长期经济落后）、无血统无游侠线、攻城武器无加成、对经济型文明长期劣势。

### 2. Franks（法兰克）

**设计哲学**：重骑兵文明，以强力农业经济支撑骑士风暴。

**签名单位/科技**：
- 独特单位：Throwing Axeman（掷斧兵）—— 远程步兵，投掷战斧造成近战伤害
- 独特科技：Chivalry（城堡，马厩 +40% 生产速度）、Bearded Axe（帝王，掷斧兵 +1 射程）
- 文明加成：骑兵 +20% HP、农场升级免费（Horse Collar / Heavy Plow / Crop Rotation）、浆果丛 +25% 食物、城堡 -25% 石料

**角色分工**：

| 角色 | 承担单位 |
|---|---|
| 步兵 | Throwing Axeman（远程步兵）、Champion（全科技） |
| 弓兵 | Arbalester（无拇指环）、无 Elite Skirmisher 升级 |
| 骑兵 | Paladin（+20% HP = 192 HP 超肉游侠）、Knight（城堡核心） |
| 僧侣 | Monk（全科技但无加成） |
| 攻城 | Trebuchet、Bombard Cannon（无 Siege Onager） |
| 海军 | Fast Fire Ship、Cannon Galleon（一般） |

**核心协同组合**：
1. 免费农场升级 + 浆果 +25% = 极速食物经济启动
2. +20% HP Knight = 城堡时代最强骑士（120 HP vs 标准 100）
3. 城堡 -25% 石料 = 早期城堡压制 + 掷斧兵驻军
4. Throwing Axeman + Knight = 远程步兵 + 重骑兵混合（反长枪 + 反弓）
5. Paladin + Trebuchet = 帝王时代推塔组合

**强势时间窗**：城堡时代（骑士风暴）、帝王时代（Paladin 192 HP 碾压）

**反制弱点**：弓兵线弱（无拇指环、无 Bracer）、海军一般、攻城武器无 Siege Onager、无血统（但 +20% HP 弥补）。

### 3. Mongols（蒙古）

**设计哲学**：游牧突袭文明，以狩猎经济和高速机动构筑全图压制。

**签名单位/科技**：
- 独特单位：Mangudai（蒙古突骑）—— 骑射手，对攻城武器有额外伤害
- 独特科技：Nomads（城堡，房屋被毁后仍提供人口）、Drill（帝王，攻城武器 +50% 移速）
- 文明加成：猎人 +40% 工作速度、轻骑兵线 +30% HP、骑射手 +20% 射速

**角色分工**：

| 角色 | 承担单位 |
|---|---|
| 步兵 | Champion（无 Final upgrades 限制）、Halberdier |
| 弓兵 | Cavalry Archer / Heavy Cav Archer（+20% 射速）、Arbalester（无拇指环） |
| 骑兵 | Hussar（+30% HP = 97 HP 超肉轻骑）、Paladin（无血统） |
| 攻城 | Siege Ram / Onager / Heavy Scorpion（Drill 后 +50% 移速） |
| 海军 | 一般 |

**核心协同组合**：
1. 猎人 +40% = 极速封建/城堡升级（最快经济启动）
2. Mangudai + Drill Siege = 反攻城骑射 + 高速冲车/投石机
3. +30% HP Hussar = 超肉轻骑兵海（反僧侣 + 突袭）
4. +20% 射速 Cav Archer = 高 DPS 骑射（0.8s/箭 vs 标准 2.0s... 实际约 1.67s/箭）
5. Hussar + Mangudai + Drill Onager = 帝王全能机动军团

**强势时间窗**：封建（快速升级骚扰）、城堡（Mangudai + 围攻）、帝王（Hussar + Mangudai + Drill Siege 全能）

**反制弱点**：无步兵加成、经济加成仅限狩猎（有限资源）、无海军加成、无建筑加成。

### 4. Japanese（日本）

**设计哲学**：步兵文明，以快速采集和渔船经济构筑海陆双线优势。

**签名单位/科技**：
- 独特单位：Samurai（武士）—— 快攻速步兵，对独特单位有额外伤害
- 独特科技：Yasama（城堡，塔 +2 额外箭矢）、Kataparuto（帝王，投石机 +25% 射速 + 快速展开）
- 文明加成：渔船 2x HP / +2 远程护甲 / 每时代 +5% 工作速度、资源收集建筑 -50% 木材、步兵 +25% 攻速（封建起）

**角色分工**：

| 角色 | 承担单位 |
|---|---|
| 步兵 | Champion（+25% 攻速 = 步兵 DPS 碾压）、Samurai（反独特单位）、Halberdier |
| 弓兵 | Arbalester（全科技）、Elite Skirmisher |
| 骑兵 | Light Cavalry / Hussar（无血统、无游侠） |
| 攻城 | Trebuchet（Kataparuto 加成）、Bombard Cannon、Onager |
| 海军 | Galleon + 渔船经济（海图强势） |

**核心协同组合**：
1. 渔船经济 + 资源建筑 -50% 木材 = 海图极速经济
2. +25% 攻速 Champion = 步兵 DPS 碾压（实际攻速 1.6s vs 标准 2.0s）
3. Samurai + 快攻速 = 反独特单位杀手（+5 vs 独特单位）
4. Yasama 塔 + 步兵防守 = 塔防阵地（每塔多 2 箭矢）
5. Kataparuto Trebuchet + 步兵保护 = 快速攻城推进

**强势时间窗**：封建（M@A rush）、城堡（Samurai + 弓兵）、帝王（Champion + Kataparuto Trebuchet）

**反制弱点**：无骑兵加成（无游侠）、无经济加成（仅限海图）、攻城武器一般（无 Siege Onager）。

### 5. Britons（不列颠）

**设计哲学**：长弓文明，以超远射程和牧羊经济构筑弓兵阵地战优势。

**签名单位/科技**：
- 独特单位：Longbowman（长弓兵）—— 超长射程弓兵（6 射程，Yeomen+Bracer 后达 12）
- 独特科技：Yeomen（城堡，步弓手 +1 射程）、Warwolf（帝王，投石机获得溅射伤害）
- 文明加成：城镇中心 -50 木材、弓兵城堡时代 +1 射程 / 帝王再 +1（共 +2）、牧羊人 +25% 工作速度

**角色分工**：

| 角色 | 承担单位 |
|---|---|
| 步兵 | Champion（全科技）、Halberdier |
| 弓兵 | Longbowman（核心，12 射程）、Arbalester（7 射程）、Elite Skirmisher |
| 骑兵 | Light Cavalry / Hussar（无血统、无游侠） |
| 攻城 | Trebuchet（Warwolf 溅射）、Bombard Cannon（无 Siege Onager） |
| 海军 | 一般 |

**核心协同组合**：
1. 牧羊人 +25% = 稳定食物经济启动
2. Crossbowman +1 射程 = 城堡时代射程碾压（6 vs 标准 5）
3. Longbowman + Yeomen + Bracer = 12 射程超远程（碾压一切弓兵对射）
4. TC -50W = 多 TC 经济扩张
5. Warwolf Trebuchet + Longbowman = 溅射攻城 + 超远程掩护

**强势时间窗**：城堡时代（Crossbow +1 射程 power spike）、帝王时代（Longbowman 12 射程碾压）

**反制弱点**：无骑兵加成（无游侠）、无步兵加成、攻城武器一般、对冲车突脸时弓兵无力、轻骑兵突袭可绕过长弓阵地。

### 6. Vikings（维京）

**设计哲学**：步兵+海军文明，以免费经济科技和步兵血量加成构筑中前期优势。

**签名单位/科技**：
- 独特单位：Berserk（狂战士）—— 可自我回血的步兵
- 独特单位：Long Boat（长船）—— 发射多支箭矢的战舰
- 独特科技：Berserkergang（城堡，狂战士回血速度 +50%）、Chieftains（帝王，所有步兵 +2 对骑兵附加伤害）
- 文明加成：战舰 -15%/-20% 造价（封建/城堡+帝王）、步兵每时代 +10%/+15%/+20% HP（封建/城堡/帝王）、免费 Wheelbarrow + Hand Cart

**角色分工**：

| 角色 | 承担单位 |
|---|---|
| 步兵 | Berserk（自愈步兵，20% HP 加成）、Champion（全科技）、Halberdier |
| 弓兵 | Arbalester（无拇指环）、Elite Skirmisher |
| 骑兵 | 无 Knight 线（仅 Scout → Light Cav）、无血统 |
| 攻城 | Onager、Bombard Cannon（无 Siege Onager） |
| 海军 | Long Boat（核心海军）、Galleon、Fire Ship |

**核心协同组合**：
1. 免费 Wheelbarrow + Hand Cart = 免费经济科技（省 350F 175W）
2. +20% HP Champion = 帝王步兵 84 HP（vs 标准 70）
3. Berserk + Berserkergang = 自愈步兵军团（无需僧侣治疗）
4. 战舰 -20% + Long Boat = 海军优势（低成本战舰海）
5. Chieftains + Berserk/Champion = 步兵反骑兵能力（+2 vs 骑兵）

**强势时间窗**：封建（免费经济科技）、城堡（步兵 + Long Boat）、帝王中期（Berserk + Arbalest）

**反制弱点**：无 Knight 线（严重缺骑兵）、无血统、无火药单位、攻城武器一般、帝王后期相对弱势（无游侠/火药）。

### 7. Chinese（中国）

**设计哲学**：万能型文明，以多村民开局和科技折扣构筑全线科技优势。

**签名单位/科技**：
- 独特单位：Chu Ko Nu（诸葛弩）—— 连弩手，一次发射多支箭矢
- 独特科技：Great Wall（城堡，城墙和塔 +30% HP）、Rocketry（帝王，诸葛弩 +2 攻击、弩车 +4 攻击）
- 文明加成：开局 +3 村民、-200 食物 -50 木材、科技每时代 -10%/-15%/-20% 成本（封建/城堡/帝王）、城镇中心 +5 人口、爆破船 +50% HP

**角色分工**：

| 角色 | 承担单位 |
|---|---|
| 步兵 | Champion（全科技）、Halberdier |
| 弓兵 | Chu Ko Nu（核心连弩）、Arbalester、Elite Skirmisher |
| 骑兵 | Knight → Paladin（无血统）、Light Cavalry / Hussar |
| 攻城 | Siege Onager、Heavy Scorpion（Rocketry 加成）、Bombard Cannon |
| 海军 | Demolition Ship（+50% HP）、Galleon、Cannon Galleon |

**核心协同组合**：
1. +3 村民 = 极速经济启动（6 村民 vs 标准 3）
2. 科技 -20% 成本 = 帝王时代全线科技折扣
3. Chu Ko Nu + Rocketry = 连弩 2+4=6 攻击/箭 × 多箭 = 极高 DPS
4. TC +5 人口 = 不用早建房屋，省木材
5. Demo Ship +50% HP = 海上自杀船威胁（75 HP vs 标准 50）

**强势时间窗**：黑暗时代（经济优势启动）、城堡时代（Chu Ko Nu）、帝王时代（全科技 + Rocketry）

**反制弱点**：开局资源紧张（-200F -50W，早期需微操经济）、无直接军事加成、依赖科技优势而非单体碾压。

### 8. Spanish（西班牙）

**设计哲学**：火药+僧侣文明，以贸易经济和火药优势构筑后期统治力。

**签名单位/科技**：
- 独特单位：Conquistador（征服者）—— 骑马火枪手，高机动远程
- 独特单位：Missionary（传教士）—— 骑马僧侣，移速快但射程短
- 独特科技：Inquisition（城堡，僧侣招降速度加快）、Supremacy（帝王，村民战斗力大幅提升）
- 文明加成：建筑工人 +30% 速度、铁匠铺升级无需黄金、炮舰获得弹道学（可攻击移动目标）

**角色分工**：

| 角色 | 承担单位 |
|---|---|
| 步兵 | Champion（全科技）、Halberdier |
| 弓兵 | Hand Cannoneer（无弓兵线，无 Arbalester）、Elite Skirmisher |
| 骑兵 | Conquistador（核心骑马火枪）、Paladin（全科技含血统）、Hussar |
| 僧侣 | Missionary（骑马僧侣）、Monk（Inquisition 加速招降） |
| 攻城 | Bombard Cannon、Trebuchet（无 Siege Onager） |
| 海军 | Cannon Galleon（+Ballistics 精准炮击）、Galleon |

**核心协同组合**：
1. 贸易 +25% 黄金（团队加成）= 后期无限黄金经济
2. Conquistador + 免费铁匠铺升级 = 高机动火药骑兵（11 攻击/枪）
3. Missionary + Inquisition = 快速招降骑兵军团
4. Cannon Galleon + Ballistics = 海上精准移动炮击
5. Supremacy 村民 = 后期村民变战士（+6 攻击 +2 护甲）

**强势时间窗**：城堡时代（Conquistador rush）、帝王时代（贸易经济 + 火药 + Paladin）

**反制弱点**：无弓兵线（无拇指环、无 Arbalester）、前期经济一般（无经济加成）、无 Siege Onager、依赖黄金单位（无垃圾兵优势）。
