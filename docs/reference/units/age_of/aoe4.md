## 第一部分：文明设计分析

### 1. English（英格兰）

**设计哲学**：防御型弓兵文明，以长弓和城堡网络构筑阵地战优势。

**签名机制**：
- **Network of Castles（城堡网络）**：城镇中心、哨塔、塔楼、石墙附近的英格兰单位获得攻击速度加成（最高 +25%），层层叠叠的防御建筑形成"主场作战"加成。
- 农场更便宜（-50% 木材），且农场采集速度更快。
- 城镇中心自动射击箭矢。
- 弓兵射程随时代 +1（封建开始）。

**四阶段角色分工**：
- 黑暗：以长弓兵（Longbowman）进行早期压制，农场经济起步。
- 封建：长弓兵 + 木墙阵地推进，Wynguard 机制开始发力。
- 城堡：国王之厅（King's Hall）产兵，Network of Castles 防守反击。
- 帝王：长弓射程极远 + 箭矢齐射（Arrow Volley）科技，Wynguard 精锐部队。

**核心协同组合**：
1. 长弓兵 + 木墙/石墙阵地（Palisade + Longbow push）
2. 长弓兵 + 投石机（Trebuchet）攻城推进
3. 国王之厅 Wynguard 部队 + Network of Castles 防守
4. 重甲步兵（Man-at-Arms）+ 长弓兵反骑兵
5. 城堡网络 + 城镇中心火力叠加防守

**文明加成 + 独特单位/科技**：
- 加成：农场便宜 50%、弓兵射程 +1/时代、城镇中心射箭、木墙更强
- 独特单位：Longbowman（长弓兵）、King（国王，仅 King of the Hill 模式）、Wynguard 部队（通过 King's Hall 地标生产）
- 独特科技：Arrow Volley（长弓兵齐射）、Wynguard Ranger / Wynguard Guard 等

**强势时间窗**：封建时代（长弓兵压制）、帝王时代（远程优势最大化）

**反制弱点**：骑兵侧翼突袭绕过长弓阵地、重型攻城武器（Bombard）摧毁建筑网络、机动性差难以追击。

### 2. French（法兰西）

**设计哲学**：骑兵文明，以皇家骑士和贸易构筑经济与军事双重压制。

**签名机制**：
- **Royal Bloodlines（皇家血统）**：所有骑兵单位获得 HP 加成（+20% 或类似），骑士系在封建时代即可登场。
- 贸易站产出更多黄金。
- 铁匠铺科技成本更低且研究更快，影响骑兵战斗力。
- 法兰西村民采集更快。

**四阶段角色分工**：
- 黑暗：贸易经济起步，铁匠铺科技准备。
- 封建：Royal Knight（皇家骑士）骑袭，村民经济领先。
- 城堡：骑士 + 弩手（Arbalétrier）组合，Keep 地标产皇家火炮。
- 帝王：皇家火炮（Royal Cannon）+ 精锐骑士 + 弩手全面军力。

**核心协同组合**：
1. 皇家骑士 + 铁匠铺科技（早期骑袭）
2. 皇家骑士 + 弩手（Arbalétrier）反步兵
3. 皇家火炮 + 骑士保护攻城
4. 贸易经济 + 骑士海
5. Keep 地标 + 骑士快速响应防守

**文明加成 + 独特单位/科技**：
- 加成：骑兵 HP +20%、贸易产出更高、铁匠铺科技更便宜更快、村民采集更快
- 独特单位：Royal Knight（皇家骑士）、Arbalétrier（重型弩手）、Royal Cannon（皇家火炮）
- 独特科技：Chivalry（骑士移速）、Gregorian（僧侣速度）

**强势时间窗**：封建（皇家骑士压制）、城堡（骑士+弩手组合成型）

**反制弱点**：大量长矛兵（Spearmen）克制骑兵、骆驼兵反制、攻城武器对密集骑兵群有效、经济依赖贸易线可被骚扰。

### 3. Holy Roman Empire（神圣罗马帝国）

**设计哲学**：宗教步兵文明，以神父加成经济和圣物积累构筑中后期优势。

**签名机制**：
- **Prelate（神父）**：可 Inspire（激励）村民，采集速度 +40%，是 HRE 经济的核心。
- **Relic（圣物）系统**：圣物产出更多黄金，且为军事单位提供伤害加成。
- 黑暗时代即可通过 Meinwerk Palace 地标生产重甲步兵（Man-at-Arms）。

**四阶段角色分工**：
- 黑暗：Prelate 加成经济，Meinwerk Palace 出 MAA。
- 封建：MAA + 弓兵推进，Prelate 持续激励经济。
- 城堡：Landsknecht（双手剑士）+ 弩手，圣物经济滚雪球。
- 帝王：圣物伤害加成 + 重甲步兵海 + Burgrave Palace 批量生产。

**核心协同组合**：
1. Prelate 激励村民 + 经济碾压
2. 黑暗 MAA（Meinwerk）+ 弓兵推进
3. Landsknecht + 弩手（反骑兵 + 反步兵）
4. 圣物黄金 + 伤害加成滚雪球
5. Burgrave Palace 批量 MAA 生产线

**文明加成 + 独特单位/科技**：
- 加成：Prelate 激励采集 +40%、圣物黄金更多 + 伤害加成、MAA 可黑暗时代出现
- 独特单位：Prelate（神父）、Landsknecht（双手剑士，溅射伤害）
- 独特科技：Marching Drills（民兵移速）、Two-Handed Weapons、Piety

**强势时间窗**：城堡时代（Landsknecht + 圣物经济）、帝王时代（伤害加成满级）

**反制弱点**：早期快攻可打乱 Prelate 激励节奏、圣物被抢夺则经济受损、Landsknecht 无甲易被弓兵集火、机动性不足。

### 4. Chinese（中国）

**设计哲学**：王朝文明，以朝代切换和火药武器构筑多功能军力。

**签名机制**：
- **Dynasty System（朝代系统）**：建造双地标进入朝代（唐→宋→元→明），每个朝代提供独特加成和单位。
  - 唐：+1 侦察兵视野
  - 宋：农场食物采集 +50%
  - 元：村民 HP +20% 和速度 +1
  - 明：军事单位 HP +20%
- **Imperial Official（朝廷官员）**：监督建筑（生产 +150%）或收税（Tax → 黄金）。
- 火药武器文明：Fire Lance、Grenadier、Nest of Bees。

**四阶段角色分工**：
- 黑暗：朝廷官员监督经济，唐朝加成探索。
- 封建：宋朝农场经济爆发，Zhuge Nu（诸葛弩）防守。
- 城堡：元朝村民增强，火药部队开始出现。
- 帝王：明朝军事 HP +20%，Nest of Bees + Grenadier 火药海。

**核心协同组合**：
1. 朝廷官员监督 + 农场经济（宋朝）
2. Nest of Bees（蜂巢箭）+ 弩手（远程压制）
3. Grenadier（掷弹兵）+ Palace Guard（禁卫军）
4. Fire Lance（火枪）+ 骑兵冲锋
5. 朝代切换 + 朝廷官员收税黄金流

**文明加成 + 独特单位/科技**：
- 加成：朝代系统（多层加成）、朝廷官员监督/收税、火药武器齐全
- 独特单位：Imperial Official、Zhuge Nu（诸葛弩）、Fire Lance（火枪）、Grenadier（掷弹兵）、Nest of Bees（蜂巢箭）、Palace Guard（禁卫军）
- 独特科技：各朝代独特科技、Spirit Way 地标

**强势时间窗**：帝王时代（明朝 +20% HP + 火药海）、宋朝经济爆发期

**反制弱点**：早期朝代未成型时脆弱、需管理双地标成本、朝代切换期间有真空期、火药单位造价高。

### 5. Mongol（蒙古）

**设计哲学**：游牧文明，以可移动建筑和早期骑袭构筑全图压制。

**签名机制**：
- **可移动建筑**：除地标外所有建筑可打包移动，全图灵活部署。
- **Ovoo（敖包）**：产出石料，可用于双倍生产单位或双倍科技研究。
- **Pasture（牧场）**：以养羊代替农场，不依赖农田。
- 起始即满人口（无需建房屋）。
- 不能建造石墙（仅有木栅栏）。

**四阶段角色分工**：
- 黑暗：Khan（可汗）+ 骑袭，Ovoo 双倍生产。
- 封建：Mangudai（蒙古突骑）骑射骚扰，银树地标贸易。
- 城堡：Keshik（重骑兵）+ Mangudai 机动军团。
- 帝王：移动基地突袭 + 双倍生产暴兵。

**核心协同组合**：
1. Khan + Mangudai 骑射骚扰（早期）
2. Ovoo 双倍生产 + 暴兵
3. Keshik 重骑兵 + Mangudai 骑射（机动军团）
4. 银树地标 + 贸易经济
5. 移动建筑 + 突袭基地前压

**文明加成 + 独特单位/科技**：
- 加成：建筑可移动、起始满人口、Ovoo 产石料、牧场代农场
- 独特单位：Khan（可汗，英雄单位）、Mangudai（蒙古突骑）、Keshik（重骑兵）
- 独特科技：Raid Bounty（攻击建筑产金）、Whistling Arrows（可汗箭矢技能）

**强势时间窗**：黑暗/封建（骑袭压制）、Ovoo 双倍生产窗口

**反制弱点**：无石墙防御脆弱、经济依赖牧场和劫掠、正面阵地战弱、被封锁地图则机动优势消失。

### 6. Rus（罗斯）

**设计哲学**：混合文明，以狩猎经济和宗教加成构筑早期到中期的全面优势。

**签名机制**：
- **Bounty Hunting（赏金狩猎）**：猎杀动物产出黄金，积累赏金等级获得全局经济加成。
- **Wooden Fortress（木堡）**：集哨塔与防御于一体的独特建筑。
- **Warrior Monk（武僧）**：可战斗并可祝福单位。
- **Gilded Units（镀金单位）**：可生产镀金版本的重骑兵、骑射手、弩手，属性更强。

**四阶段角色分工**：
- 黑暗：狩猎经济爆发，赏金积累黄金。
- 封建：镀金骑士 + 弓兵，木堡防守。
- 城堡：Warrior Monk 祝福 + 镀金部队，骑士 HP +20%。
- 帝王：镀金骑射手 + 镀金弩手 + 重骑兵全面军力。

**核心协同组合**：
1. 狩猎经济 + 赏金黄金流
2. 镀金骑士 + Warrior Monk 祝福
3. 木堡防守 + 镀金弩手反击
4. 镀金骑射手 + 重骑兵机动军团
5. Lodya 船只（海军优势）

**文明加成 + 独特单位/科技**：
- 加成：赏金狩猎（猎物产金）、骑士城堡时代 HP +20%、木堡防御、农场更便宜但更慢
- 独特单位：Warrior Monk（武僧）、Gilded Knight（镀金骑士）、Gilded Horse Archer（镀金骑射手）、Gilded Crossbowman（镀金弩手）、Lodya（独特船只）
- 独特科技：Blessing of the Forest、Knight Templar

**强势时间窗**：黑暗/封建（狩猎经济领先）、城堡（镀金骑士 + 武僧）

**反制弱点**：农场采集慢影响后期经济、赏金用尽后经济回落、镀金单位造价高、无强力火药单位。

### 7. Delhi Sultanate（德里苏丹国）

**设计哲学**：科研文明，以学者免费研究科技和战象构筑后期碾压。

**签名机制**：
- **Scholar（学者）**：可分配到建筑中免费研究科技，学者越多研究越快。
- **Sacred Site（圣地）**：步兵可占领圣地获得被动黄金。
- **战象**：War Elephant（战象）和 Tower Elephant（塔象），高 HP 碾压型单位。
- 科技免费但研究慢，需大量学者加速。

**四阶段角色分工**：
- 黑暗：学者准备科技，步兵探索圣地。
- 封建：免费科技优势开始显现，步兵占圣地。
- 城堡：战象出场，Tower Elephant 远程压制。
- 帝王：免费科技全部完成 + 战象海碾压。

**核心协同组合**：
1. 学者免费科技 + 经济领先
2. 步兵占领圣地 + 被动黄金
3. War Elephant + Tower Elephant（象兵碾压）
4. Tower Elephant（远程）+ 步兵保护
5. 清真寺地标 + 学者加速研究

**文明加成 + 独特单位/科技**：
- 加成：科技免费（需学者加速）、步兵占圣地、战象单位
- 独特单位：Scholar（学者）、War Elephant（战象）、Tower Elephant（塔象）
- 独特科技：Tower of Victory（圣地/圣物黄金加成）

**强势时间窗**：帝王时代（免费科技满级 + 战象海）

**反制弱点**：早期科技研究慢、学者被杀则研究停滞、战象造价极高且被长矛兵/骆驼克制、早期军力薄弱。

### 8. Ottoman（奥斯曼）

**设计哲学**：军事生产文明，以军事学校免费产兵和维齐尔点数构筑持续军力。

**签名机制**：
- **Military School（军事学校）**：自动免费生产军事单位（Sipahi、Janissary 等），无需资源。
- **Vizier Points（维齐尔点数）**：随时间积累，可花费在独特加成上。
- **Janissary（耶尼切里）**：强力火药步兵。
- **Mehter（军乐队）**：音乐家单位，为附近军事单位提供 buff。

**四阶段角色分工**：
- 黑暗：军事学校开始免费产兵，维齐尔点数积累。
- 封建：Sipahi 轻骑兵 + 军事学校持续供给。
- 城堡：Janissary 火枪兵 + Mehter buff。
- 帝王：Grand Bombard（巨型火炮）+ Janissary 海 + 军事学校暴兵。

**核心协同组合**：
1. 军事学校免费产兵 + 经济不冲突
2. Janissary + Mehter（火枪 + buff）
3. Grand Bombard + Janissary 攻城推进
4. Sipahi 轻骑 + 维齐尔加成
5. 军事学校多线 + 持续暴兵

**文明加成 + 独特单位/科技**：
- 加成：军事学校免费产兵、维齐尔点数系统、军乐队 buff
- 独特单位：Janissary（耶尼切里火枪兵）、Sipahi（西帕希轻骑兵）、Grand Bombard（巨型火炮）、Mehter（军乐队）
- 独特科技：Janissary Volley、Mehter buff 系列科技

**强势时间窗**：城堡/帝王（Janissary + 军事学校持续产出）、后期 Grand Bombard

**反制弱点**：军事学校产兵类型固定可被针对、早期军事学校覆盖前薄弱、Janissary 射速慢被骑兵近身克制。

### 9. Malians（马里）

**设计哲学**：经济文明，以矿井采金和牲畜系统构筑灵活经济与隐蔽军力。

**签名机制**：
- **Pit Mine（矿井）**：在矿石上建造，被动产出黄金。
- **Cattle（牛群）**：可指派到房屋产出食物，替代传统农场。
- **Musofadi Warrior（隐身步兵）**：可隐身潜入敌方阵地的特殊步兵。
- **Farima（领袖）**：可集结部队的英雄单位。

**四阶段角色分工**：
- 黑暗：矿井采金经济起步，牛群食物准备。
- 封建：Musofadi 隐身骚扰 + Javelin Thrower。
- 城堡：Sofa 骑兵 + Farima 领袖 buff。
- 帝王：Griot 吟游诗人 + 隐身突袭 + 矿井经济。

**核心协同组合**：
1. 矿井黄金 + 牛群食物（双经济流）
2. Musofadi 隐身突袭 + 骚扰经济
3. Farima + Sofa 骑兵（领袖 buff 军团）
4. Javelin Thrower + Musofadi（反骑兵 + 隐身）
5. Griot + 多线骚扰

**文明加成 + 独特单位/科技**：
- 加成：矿井被动产金、牛群产食物、隐身步兵
- 独特单位：Musofadi Warrior（隐身步兵）、Farima（领袖）、Sofa（骑兵）、Griot（吟游诗人）、Javelin Thrower（标枪手）
- 独特科技：Cattle Ranches、Mansa Quarry

**强势时间窗**：封建（Musofadi 隐身骚扰）、中后期（矿井经济 + Farima 军团）

**反制弱点**：正面阵地战弱、隐身单位被侦察克制、缺少重型攻城武器、军力单薄需靠骚扰。

### 10. Abbasid Dynasty（阿拔德王朝）

**设计哲学**：黄金时代文明，以智慧宫和骆驼兵构筑科技与反骑兵优势。

**签名机制**：
- **House of Wisdom（智慧宫）**：唯一地标，通过升级翼（Wing）推进时代，不建造传统地标。
- **Golden Age（黄金时代）**：根据建筑数量进入黄金时代层级，提供采集速度、研究速度等加成。
- **Camel Units（骆驼单位）**：Camel Rider 和 Camel Archer，反骑兵专精。
- 可在沙地上建造。

**四阶段角色分工**：
- 黑暗：智慧宫翼升级准备，黄金时代经济起步。
- 封建：Camel Rider 反骑兵 + 弓兵。
- 城堡：Camel Archer + 黄金时代经济滚雪球。

## 附录：AoE4 Bonus Damage 系统说明

### 单位标签（Tags）

AoE4 的单位拥有以下主要标签，用于判定 bonus damage：

| 标签 | 说明 | 典型单位 |
|---|---|---|
| Heavy | 重型 | MAA、Knight、Landsknecht |
| Light | 轻型 | Spearman、Archer、Horseman |
| Cavalry | 骑兵 | Knight、Horseman、Camel |
| Infantry | 步兵 | 所有步行单位 |
| Ranged | 远程 | Archer、Crossbowman、Mangonel |
| Siege | 攻城 | 所有攻城武器 |
| Gunpowder | 火药 | Handcannoneer、Bombard、Janissary |
| Building | 建筑 | 所有建筑 |
| Camel | 骆驼 | Camel Rider、Camel Archer |
| Elephant | 象 | War Elephant、Tower Elephant |
| Naval | 海军 | 所有战船 |
| Stealth | 隐身 | Musofadi Warrior、Shinobi |
| Gilded | 镀金（Rus） | Gilded Knight 等 |
| Samurai | 武士 | Samurai |
| Hero | 英雄 | Khan、Daimyo、Farima |

### Bonus Damage 计算公式

```
最终伤害 = 基础伤害 + bonus_damage(对目标标签) - 目标护甲
```

### 主要 Bonus Damage 关系

| 攻击单位 | 对目标标签 | Bonus Damage |
|---|---|---|
| Spearman | Cavalry | +8 |
| Crossbowman | Heavy | +5 |
| Handcannoneer | Heavy | +4 |
| Janissary | Heavy | +4 |
| Camel Rider | Cavalry | +5 |
| Camel Archer | Cavalry | +5 |
| Springald | Siege, Naval | +bonus |
| Bombard | Siege, Building | +bonus |
| Mangonel | Ranged | +bonus |
| Musofadi Warrior | Cavalry | +6 |
| Javelin Thrower | Cavalry | +8 |
| Limitanei | Cavalry | +6 |
| Demolition Ship | Naval | 高额溅射 |

### 升级系统说明

| 升级阶段 | 可用时代 | 典型成本 | 效果 |
|---|---|---|---|
| Base（基础） | 生产时代 | — | 基础属性 |
| Veteran（老兵） | 城堡时代 | ~75F 75G | HP +20~30，护甲 +1 |
| Elite（精锐） | 帝王时代 | ~150F 150G | HP +20~30，伤害 +2~3，护甲 +1 |

**注意**：
- 城堡时代生产的单位（如 Knight）只有 Elite 升级（帝王时代）。
- 部分独特单位有独特升级名称（如 French 的 Royal Bloodlines 科技影响所有骑兵）。
- 中国朝代系统、HRE 圣物加成等文明机制可额外提升单位属性。
