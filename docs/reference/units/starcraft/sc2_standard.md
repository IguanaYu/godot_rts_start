# 星际争霸2 标准对战单位参考

> 数据版本: SC2 LotV 5.0+ / 数据源: Liquipedia + fandom / 采集日期: 2026-06-17

## 第一部分：种族设计分析

---

### Terran（人族）

#### 1. 设计哲学
**"先造后出、机动换防、火力至上"** — Terran 以高 DPS 玻璃炮（Glass Cannon）为核心，依赖阵地推进和机动空投进行战术展开，单位以"建造→空投→架起"三段式为特征。

#### 2. 核心签名机制
- **飞挂（Lift-Off）**：建筑可起飞移动，用于逃生/侦察/堵路
- **空投（Medivac / Dropship）**：主力运输+治疗，是 Terran 战术核心
- **架起/展开（Siege / Transform）**：Siege Tank / Liberator / Thor 等有双形态
- **修理（SCV Repair）**：SCV 可修理机械单位/建筑，提供持续作战
- **Add-on 系统**：Barracks / Factory / Starport 可挂载 Tech Lab / Reactor
- **MULE / Scan / Calldown**：轨道指挥部（Orbital Command）的三技能

#### 3. 角色分工矩阵

| 层级 | 肉盾 | 近战输出 | 远程输出 | 法师/功能 | 攻城 | 骚扰 | 侦察 | 防空 | 运输 | 经济 |
|------|------|----------|----------|-----------|------|------|------|------|------|------|
| **T1** | Marine | — | Marine | — | — | Reaper | — | Marine | — | SCV |
| **T2** | Marauder | Hellion | Marauder / Cyclone | Ghost / Raven | Siege Tank | Hellion / Banshee / Widow Mine | Raven | Viking | Medivac | — |
| **T3** | Thor | Hellbat | Battlecruiser | — | Liberator | Liberator | — | Thor / Viking | — | — |

#### 4. 科技树节奏

| 阶段 | 时间窗 | 标志性 Timing |
|------|--------|---------------|
| 早期（0-4:00） | 1-1-1 开局 | Reaper 侦察 / 2-1-0 死神骚扰 |
| 中期（4:00-8:00） | Stim + Combat Shield 完成 | 两船兵空投 / 坦克推进 |
| 中后期（8:00-12:00） | +2/+2 升级 | Ghost EMP 拆盾 / Viking 控空 |
| 后期（12:00+） | 3-3 满攻防 | Battlecruiser 主力 / Raven 干扰 |

#### 5. 核心协同组合

| 组合名称 | 构成 | 类型 | 说明 |
|----------|------|------|------|
| **MMM (Bio Ball)** | Marine + Marauder + Medivac | 主力正面 | 最经典的 Terran 组合，Stim 后高爆发 |
| **Mech (机械化)** | Hellion/Cyclone/Siege Tank/Thor | 主力正面 | 重甲高耐久，靠阵地推进 |
| **Sky Terran** | Viking + Liberator + Battlecruiser | 纯空军 | 后期制空权，BC 肉盾 + Liberator 输出 |
| **Ghost Rush** | Ghost + EMP | 战术组合 | 针对 Protoss 护盾 / Zerg 能量 |
| **Bio + Mine** | Marine + Widow Mine | 防守/反制 | Widow Mine 埋地防 Zergling 包夹 |
| **Tank + Viking** | Siege Tank + Viking | 阵地战 | 坦克架高坡 + Viking 提供视野/防空 |
| **Hellbat + Medivac** | Hellbat + Medivac | 空投骚扰 | 空投 Hellbat 烧工兵 |
| **Raven + Mech** | Raven + 机械化 | 辅助 | Raven 提供干扰矩阵+自动炮台 |
| **BC Teleport** | Battlecruiser (Tactical Jump) | 奇袭 | 跳脸主矿秒杀经济 |
| **Cyclone + Hellion** | Cyclone + Hellion | 中期压制 | 高速机动部队抓图 |

#### 6. 生产机制
- **单线生产**：Barracks / Factory / Starport 各自独立生产队列
- **Add-on 系统**：Tech Lab（解锁高级单位/升级）vs Reactor（双倍训练速度）
- **Swapping（建筑交换）**：Add-on 可在同类型建筑间互换
- **Chronoboost 不可用**（与 Protoss 区别）
- **无幼虫机制**（与 Zerg 区别）

#### 7. 机动哲学
- 主力部队（Bio）移动速度快，靠 Medivac 空投实现跨地形机动
- 机械化部队（Mech）地面慢，靠阵地推进
- 利用 Lift-Off 实现建筑逃生、高地侦察
- Scan 提供实时视野，无需单位到达

#### 8. 视野与地图控制
- **Scan（轨道指挥部）**：即时视野，CD 约 60 秒
- **Raven（侦察机）**：可移动侦察 + 干扰
- **Sensor Tower**：显示隐身的敌方单位动向
- **Marine 视野开阔**：基础视野 9
- **劣势**：没有永久隐身侦察单位（相比 Protoss Observer / Zerg Overseer）

#### 9. 反制弱点
- **怕 AOE**：Bio 密集站位易被 Baneling / Colossus / Storm 克制
- **Mech 机动性差**：容易被多线骚扰拖垮
- **后期能量依赖**：Ghost / Raven 依赖能量
- **经济扩张慢**：需要大量 SCV 和 Orbital Command 投资

#### 10. 强势时间窗
- **2-3 分钟**：Reaper 骚扰窗口
- **5-7 分钟**：Stim timing 两船兵空投
- **8-10 分钟**：+1/+1 机械化推进
- **12 分钟+**：3-3 Bio 或 Sky Terran 成型

#### 11. 经济模型
- 标准 3 基地运营（3 CC）
- 每基地 16 SCV 采矿 + 6 SCV 采气
- MULE 提供额外矿物收入（约 270 矿物/MULE）
- 经济节奏：1 CC → 2 CC 扩张 → 3 CC 后期
- Gas 分配：前期气体紧张（Stim / Combat Shield / 坦克），后期富余

### Protoss（星灵/神族）

#### 1. 设计哲学
**"精兵路线、高单体质量、技能碾压"** — 每个单位都是精英，依靠护盾（Shield）和技能组合以少打多，Warp Gate 实现全球投送。

#### 2. 核心签名机制
- **Warp Gate**：折跃门可在任意有能量的神族水晶塔（Pylon）范围内瞬间召兵
- **护盾（Shield）**：所有单位都有可再生的护盾
- **护盾电池（Shield Battery）**：为附近友军充盾
- **时空提速（Chronoboost）**：加速生产/升级
- **母舰核心（Mothership Core）**→ **母舰（Mothership）**：LotV 初期简化，移除母舰核心
- **力场（Force Field / Guardian Shield）**：Sentinel / 高阶技能

#### 3. 角色分工矩阵

| 层级 | 肉盾 | 近战输出 | 远程输出 | 法师/功能 | 攻城 | 骚扰 | 侦察 | 防空 | 运输 | 经济 |
|------|------|----------|----------|-----------|------|------|------|------|------|------|
| **T1** | Zealot | Zealot | — | — | — | Adept | — | — | — | Probe |
| **T2** | — | — | Stalker / Adept | Sentry | — | Adept / Oracle | Observer | Stalker / Phoenix | — | — |
| **T3** | Immortal | Dark Templar | Colossus / Tempest | High Templar / Disruptor | Colossus / Disruptor | DT / Oracle | — | Phoenix / Tempest | — | — |

#### 4. 科技树节奏

| 阶段 | 时间窗 | 标志性 Timing |
|------|--------|---------------|
| 早期（0-4:30） | 1-Gate Expand | Adept 侦察 / 先知骚扰 |
| 中期（4:30-7:30） | Warp Gate 完成 | 3-4 Gate 一波 / Blink 追猎 |
| 中后期（7:30-11:00） | Charge + Blink | Immortal + Zealot + Archon 推进 |
| 后期（11:00+） | 3-3 升级 + Storm | Carrier / Tempest 转型 |

#### 5. 核心协同组合

| 组合名称 | 构成 | 类型 | 说明 |
|----------|------|------|------|
| **IAC (Immortal + Archon + Chargelot)** | Immortal + Archon + Charge Zealot | 主力正面 | 经典的 Protoss 地面黄金组合 |
| **Stalker + Colossus** | Stalker + Colossus | 中期正面 | Colossus 烧轻甲，Stalker 点重甲 |
| **Blink Stalker + Disruptor** | Stalker + Disruptor | 高操作 | 净化球逼迫走位，Blink 追伤害 |
| **Sky Protoss (Carrier + Tempest)** | Carrier + Tempest + Phoenix | 纯空军 | 后期最强空军之一 |
| **DT Drop** | Dark Templar + Warp Prism | 骚扰 | 隐身刀兵空投秒杀工兵 |
| **Adept + Oracle** | Adept + Oracle | 早期压制 | Adept 给标记 + Oracle 启示 |
| **High Templar + Zealot** | HT + Zealot | 法术 | Storm 洗地 + Zealot 冲锋收割 |
| **Immortal + Sentry** | Immortal + Sentry | 防守反击 | Sentry 力场分割 + Immortal 点杀 |
| **Phoenix + Void Ray** | Phoenix + Void Ray | 空优 | Phoenix 抬地面 + Void Ray 拆建筑 |
| **Warp Prism Elevator** | Warp Prism + 任意地面 | 多线 | 利用折跃棱镜全球投送 |

#### 6. 生产机制
- **双模式生产**：Gateway（Gate）模式慢速生产 / Warp Gate 模式全球投送
- **Chronoboost**：Nexus 技能，加速生产/升级（200% 速度持续 20 秒）
- **折跃冷却（Warp-in Cooldown）**：每个 Gate 有独立的折跃冷却 CD
- **Pylon 能量场（Power Field）**：只能在 Pylon / Warp Prism 能量场内折跃

#### 7. 机动哲学
- **Warp Gate 全球投送**：只要有 Pylon 能量场即可折跃，无需部队行军
- **Warp Prism（折跃棱镜）**：移动能量场，支持空投+前线折跃
- **Blink（闪烁）**：Stalker 核心机动技能，翻山越岭
- **Recall（召回）**：Mothership 技能，将部队召回 Nexus
- 单位基础速度一般，靠技能弥补

#### 8. 视野与地图控制
- **Observer（观察者）**：永久隐身侦察单位
- **Oracle Revelation（启示）**：区域显隐 + 视野
- **Adept 幻象侦察**：Adept 的 Psionic Transfer 留下分身侦察
- **Hallucination（幻象）**：Sentry 可制造假单位侦察
- 视野控制能力强，但 Observer 会被反隐发现

#### 9. 反制弱点
- **怕 EMP**：Ghost 的 EMP 瞬间清零护盾，克制 Protoss 核心生存机制
- **怕大量 AOE 互怼**：Colossus / Disruptor / Storm 虽强但也怕对方 AOE
- **折跃 CD 限制**：一次折跃后需等待 CD，爆发后是真空期
- **前期兵力少**：单位单价高，前期兵力总数少，怕 rush

#### 10. 强势时间窗
- **4-5 分钟**：Adept + Oracle 双线压制
- **6-8 分钟**：Blink Stalker 追击 / Immortal 一波
- **9-11 分钟**：IAC 黄金组合成型
- **12 分钟+**：Storm + Carrier 后期制霸

#### 11. 经济模型
- 标准 2-3 基地运营
- Probe 效率与 SCV 相同
- Chronoboost 可加速经济（加速 Nexus 出 Probe）
- 护盾电池减少战损，变相省钱
- 单位单价高，Gas 消耗大，矿物容易溢出

### Zerg（虫族）

#### 1. 设计哲学
**"虫海战术、快速扩张、以量取胜"** — 用最少的投资快速铺开基地，依靠幼虫机制爆兵，用数量和质量变化应对对手。

#### 2. 核心签名机制
- **幼虫（Larva）机制**：Hatchery 自动产幼虫，每个基地 3 幼虫上限，是 Zerg 生产能力的根基
- **Inject Larva（注卵）**：Queen 技能，一次性 +3 幼虫
- **Creep（菌毯）**：Zerg 建筑需要菌毯，单位在菌毯上加速
- **Creep Tumor（菌毯肿瘤）**：Queen 可蔓延菌毯
- **Burrow（埋地）**：大部分 Zerg 单位可埋地，用于伏击/回血/隐藏
- **Regeneration（回血）**：生物单位在菌毯上自动回血

#### 3. 角色分工矩阵

| 层级 | 肉盾 | 近战输出 | 远程输出 | 法师/功能 | 攻城 | 骚扰 | 侦察 | 防空 | 运输 | 经济 |
|------|------|----------|----------|-----------|------|------|------|------|------|------|
| **T1** | Zergling | Zergling | — | Queen | — | Zergling / Baneling | Overlord / Changeling | Queen | Overlord | Drone |
| **T2** | Roach | Baneling | Hydralisk / Roach | Infestor | — | Baneling / Mutalisk | Overseer | Hydralisk / Mutalisk | — | — |
| **T3** | Ultralisk | Ultralisk | Lurker / Brood Lord | Viper / Infestor | Lurker / Swarm Host | Swarm Host | Overseer | Corruptor | — | — |

#### 4. 科技树节奏

| 阶段 | 时间窗 | 标志性 Timing |
|------|--------|---------------|
| 早期（0-4:00） | 扩展至 3 基地 | Speedling / Baneling 一波 |
| 中期（4:00-7:00） | Lair 完成 | Roach + Ravager 推进 / Mutalisk 骚扰 |
| 中后期（7:00-11:00） | Hive 完成 | Lurker / Viper 出现 |
| 后期（11:00+） | 5+ 基地满采 | Ultralisk / Brood Lord 主力 |

#### 5. 核心协同组合

| 组合名称 | 构成 | 类型 | 说明 |
|----------|------|------|------|
| **Zergling + Baneling (Ling-Bane)** | Zergling + Baneling | 主力正面 | 经典虫族地面组合，极高速高爆发 |
| **Roach + Hydralisk (Roach-Hydra)** | Roach + Hydralisk | 主力正面 | 肉盾+远程输出，简单粗暴 |
| **Roach + Ravager** | Roach + Ravager | 中期一波 | Ravager 胆汁破高坡阵型 |
| **Mutalisk (Muta)** | Mutalisk + Zergling | 骚扰/牵制 | 高速空军骚扰经济，地面配合牵制 |
| **Lurker + Hydralisk** | Lurker + Hydralisk | 阵地战 | Lurker 埋地封锁 + Hydra 防空 |
| **Ultralisk + Ling-Bane** | Ultralisk + Zergling + Baneling | 后期主力 | 最强地面正面，Ultra 当肉盾 |
| **Brood Lord + Corruptor** | Brood Lord + Corruptor | 后期空军 | Brood Lord 远程攻城 + Corruptor 护航空战 |
| **Swarm Host + Zergling** | Swarm Host + Zergling | 消耗战 | 免费单位消耗对手 |
| **Infestor + Fungal** | Infestor + 任意 | 法术辅助 | Fungal Growth 定身 + 反隐形 |
| **Viper + Abduct** | Viper + 任意地面 | 法术抓取 | Abduct 拉对手关键单位到包围圈 |

#### 6. 生产机制
- **幼虫（Larva）基础生产**：每个 Hatchery 自动产生幼虫（每 15 秒 1 只，上限 3）
- **Inject Larva**：Queen 技能每 40 秒 +3 幼虫，大幅提升产能
- **同时生产**：选中幼虫后同时拉多个单位（Shift 多选）
- **无 Add-on / 折跃机制**：Zerg 全凭幼虫数量和基地数量决定生产速度
- **Drone 转兵**：Drone 可变成建筑（牺牲工兵），也可变成战斗单位

#### 7. 机动哲学
- **Creep 加速**：单位在菌毯上 +30% 移动速度（Zergling 在菌毯上极快）
- **无运输单位**（Nydus Worm 除外）：Nydus 可跨地图传送
- **Burrow 机动**：埋地移动（部分单位）实现偷袭/撤退
- **空中单位高速**：Mutalisk / Corruptor 速度快，可快速转场
- **Overlord 被动侦察**：散布 Overlord 获得地图视野

#### 8. 视野与地图控制
- **Overlord 散布**：开局 Overlord 即可全场侦察
- **Overseer（监察者王虫）**：反隐 + 移动视野
- **Changeling（变形虫）**：潜入敌方单位中获取情报
- **Creep 视野**：菌毯蔓延提供视野
- **劣势**：无即时视野技能（如 Scan / Revelation），侦察靠兵种

#### 9. 反制弱点
- **怕多线**：Zerg 需要女王/虫后防守多线骚扰
- **怕前期压制**：经济扩张快但兵力前期薄弱
- **怕 AOE 清虫海**：Hellion / Colossus / Storm 对轻甲虫海极度有效
- **幼虫依赖**：一次部队被全歼后需重新积累幼虫
- **缺乏稳定肉盾**：Ultralisk 外，Zergling / Roach 都不算真正肉盾

#### 10. 强势时间窗
- **3-4 分钟**：Speedling + Baneling 一波
- **5-7 分钟**：Roach + Ravager 推进
- **8-10 分钟**：Mutalisk 骚扰黄金期
- **11 分钟+**：Ultralisk / Brood Lord 成型

#### 11. 经济模型
- **激进扩张**：通常 3 基地开局，后期 5-7 基地
- **Drone 与战斗单位共用幼虫**：造 Drone = 少造兵，造兵 = 少采资源
- **Queen 能量管理**：注卵 vs 铺菌毯 vs 防空
- **Gas 早期紧张**：Zergling 不需要气，但 Baneling / Roach 需要大量气
- **后期矿物富裕**：大量 Zergling 和 Drone 消耗矿物
