## 第一部分：战役专属机制概述

SC2 的四个战役各有独特的单位获取与强化机制，与标准对战（ Versus / Ladder ）显著不同。

### 1.1 Wings of Liberty (WoL) — 雇佣兵 + 科技实验室

- **雇佣兵 (Mercenaries)**：在休伯利安号 Cantina 用 Credits 购买解锁，之后每关可从 Merc Compound 直接折跃部署。雇佣兵比常规单位更强，且不要求对应的生产建筑。
- **科技实验室升级 (Armory / Lab)**：通过 Armory Console 和 Laboratory Console 解锁战役专属单位（如 Spectre、Diamondback、Goliath 等），消耗 Credits 和 Gas。
- **军械库升级 (Armory Upgrades)**：可给所有地面/空中单位增加攻击和护甲。

### 1.2 Heart of the Swarm (HotS) — 虫群进化

- **进化坑 (Evolution Pit)**：通过完成进化任务，从两个分支中选择一种进化变体（永久选择，不可更改）。
- **突变升级 (Mutation Upgrades)**：可随时切换的强化升级，每个单位有三种可选突变。
- **凯瑞甘技能树 (Kerrigan Skill Tree)**：凯瑞甘自身有独立的技能树，通过任务奖励的进化点数解锁。

### 1.3 Legacy of the Void (LotV) — 战争议会

- **战争议会 (War Council)**：每关前可从 Khalai / Nerazim / Tal'darim / Purifier 四个阵营中选择该关的单位变体。
- **每个基础单位有 3-4 种变体**，不同变体有完全不同的属性与技能。
- **太阳之矛 (Spear of Adun)** 提供全局支援技能。

### 1.4 Nova Covert Ops (Nova) — 装备槽系统

- **装备槽 (Equipment Loadout)**：Nova 和她的单位通过装备槽系统获得强化。
- 每个任务前可为 Nova 和主力单位选择不同的装备配置（武器、护甲、能力插件）。
- 战役专属单位包括装备变体版的陆战队、幽灵、坦克等。

| 战役 | 获取方式 | 代表单位 |
|---|---|---|
| WoL | Cantina → Merc Compound | War Pig, Hammer Security, Devil Dog... |
| WoL | Armory Console (升级) | Firebat, Medic, Vulture, Goliath... |
| WoL | Laboratory Console (二选一) | Spectre (vs Ghost), Science Vessel |
| HotS | Evolution Mission (二选一) | Raptor/Swarmling, Impaler/Lurker... |
| HotS | Mutation Upgrade (可切换) | 硬化甲壳, 肾上腺体... |
| LotV | War Council (阵营选择) | Adept, Blood Hunter, Vanguard... |
| Nova | Equipment Loadout | 多种武器/护甲/能力组合 |

> **免责声明**: 本手册的数据基于 SC2 战役合集 4.x 版本。实际数值可能因版本更新而略有变化。标记为 `—` 的字段表示该单位没有对应属性或数据不适用。
