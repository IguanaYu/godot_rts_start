## 一、关卡类型分类总表

首先提取共性，暴雪在三个模式中使用了以下**12种关卡类型**：

| 类型 | 核心玩法 | 代表关卡 |
|------|----------|----------|
| **标准进攻** | 建基地→造兵→消灭敌人基地 | 魔兽3黑石滚滚、星际2解放日 |
| **限时防御** | 守住据点N分钟，抵御波次攻击 | 魔兽3天灾进军、星际2零点时刻、诸神的黄昏 |
| **护送** | 保护NPC/车队从A到B | 魔兽3漫长的行军、星际2撤离行动 |
| **竞速/限时进攻** | 在时间压力下推进，或与AI竞争 | 星际2净化(竞速)、超新星(火墙逼迫) |
| **资源收集** | 采集指定数量资源，而非消灭敌人 | 星际2恶魔游乐场(熔岩涨潮中采矿) |
| **占领/控制** | 占领并守住地图上的关键点 | 星际2欢迎来到丛林、合作模式Lock&Load |
| **日夜循环** | 白天进攻/夜晚防御的交替节奏 | 星际2爆发 |
| **环境危险** | 熔岩/火墙/暴风雪等环境机制 | 恶魔游乐场(熔岩)、超新星(火墙) |
| **潜行/RPG** | 英雄单位探索，无基地建设 | 魔兽3追寻幻象、星际2毁灭低语、杜隆坦建立 |
| **地牢探险** | 迷宫式关卡，走廊+陷阱+探索 | 魔兽3萨格拉斯之墓、达拉然地牢、暗网洞穴 |
| **Boss战** | 击败单个强大敌人 | 魔兽3霜之哀伤、星际2至高无上、埃蒙陨落 |
| **拦截/摧毁** | 拦截移动目标(火车/运输船) | 星际2火车大劫案、合作模式Oblivion Express |

## 五、关卡设计规律总结

### 5.1 暴雪的关卡设计原则

1. **渐进式复杂度**：每个战役从RPG/教程开始→逐步引入基地建设→复杂组合
2. **类型交替**：相邻关卡类型绝不重复，防御→进攻→RPG→护送交替出现
3. **英雄中心**：即使有基地建设，英雄单位也是核心焦点(升级+物品+技能)
4. **叙事驱动机制**：关卡机制反映剧情（烧船、堕落、选择分支）
5. **标志性大关卡**：每个战役有1-2个"名场面"(天灾进军/诸神的黄昏/孤注一掷)
6. **分支选择**：提供二选一路线增加重玩价值(星际2尤为突出)

### 5.2 可借鉴到RTS游戏的设计模式

| 模式 | 适用场景 | 实现难度 |
|------|----------|----------|
| 标准进攻(建基地→推图) | 基础关卡，最常见 | 低 |
| 限时防御(守N分钟) | 节奏变化关卡 | 中 |
| 护送(保护NPC移动) | 增加紧张感 | 中 |
| 日夜循环(进攻↔防御) | 独特体验关卡 | 高 |
| 环境危险(熔岩/火墙) | 增加地图复杂度 | 中 |
| 竞速(与AI竞争目标) | 增加紧迫感 | 中 |
| 资源收集(采够N资源) | 变化胜利条件 | 低 |
| RPG/英雄探索(无基地) | 叙事关卡 | 高 |
| 地牢探险(迷宫式) | 特殊体验 | 高 |
| Boss战 | 章节/战役高潮 | 中 |
| 拦截(拦截移动目标) | 节奏变化 | 中 |
| 占领/控制点 | 多线操作 | 中 |

### 5.3 剧情模式关卡编排建议

参考暴雪的编排节奏（以一个10关的章节为例）：

| 关卡 | 类型 | 节奏作用 |
|------|------|----------|
| 1 | RPG/教程 | 教学基础操作 |
| 2 | 标准进攻(简单) | 引入基地建设 |
| 3 | 标准进攻(中等) | 巩固基础 |
| 4 | 防御/生存 | 节奏变化，增加紧张感 |
| 5 | 护送或环境危险 | 引入新机制 |
| 6 | 竞速/限时进攻 | 高潮前奏 |
| 7 | RPG/叙事 | 剧情转折，降低节奏 |
| 8 | 标准进攻(困难) | 应用所有已学技能 |
| 9 | 特殊机制关卡(日夜/地牢) | 独特体验 |
| 10 | 防御+Boss | 史诗最终战 |
