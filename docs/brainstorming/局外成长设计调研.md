# RTS 局外成长 (Meta-Progression) 设计调研

## 一、参考游戏

| 游戏 | 局外机制 | 核心特点 |
|------|---------|---------|
| Age of Darkness: Final Stand | Keep 枢纽 + 持久科技树 | 每局带回资源，升级建筑/单位/英雄 |
| StarCraft 2 战役 | 任务间研究点数+选择升级 | 二选一升级路径，单位解锁，剧情推进 |
| Northgard | 阵营专属成长 + 永久解锁 | 不同氏族有不同升级路线 |
| They Are Billions | 科技升级跨局保持 | 防御塔/城墙持久强化 |
| Stormgate | 战役英雄成长 + 3人合作持久进度 | RPG 式英雄升级 |
| Bad North | 领袖技能跨局保持 | Roguelite + RTS 混合 |
| Darkest Dungeon | Hamlet 小镇建设 + 多种材料 | 不同材料给不同建筑升级 |
| Hades | 冥府大厅 + 多 NPC 商店 | 统一货币，NPC 有独立故事线 |
| Loop Hero | 营地可视化建设 | 资源建造设施，营地从空地变成据点 |
| Slay the Spire | 卡组构建 | 每局选择带入的"卡牌"策略组合 |

## 二、30 种局外成长机制清单

### A. 货币与获取方式

1. **统一货币**（如星辰石）— 通关就给，简单直接
2. **多种材料**（废铁/宝石/草药/古卷）— 不同材料来源不同，卖给不同 NPC
3. **轻度拾荒**— 地图上少量散落物资箱，打精英怪偶尔掉特殊材料
4. **声望系统**— 每次通关增加与不同阵营的声望，解锁对应阵营的升级
5. **成就/挑战奖励**— "用不到 50 金币通关" → 奖励特殊货币

### B. Hub 功能建筑

6. **铁匠铺**— 永久提升单位属性（HP/ATK/DEF）
7. **军需处**— 解锁新兵种/新建筑类型
8. **法师塔**— 强化指挥官技能
9. **训练营**— 单位获得被动技能（如"攻击有 10% 几率击退"）
10. **仓库/银行**— 存金币跨关使用（比如这关省下的金下关能用一部分）
11. **炼金台**— 制作一次性消耗品（回城卷轴、紧急增援令），下一关可用
12. **黑市**— 随机刷新的稀有物品，每次通关换一批
13. **图书馆**— 解锁被动知识（如"看到敌人攻击范围"）
14. **祭坛**— 赌博性质的升级（随机给一个强力 buff 或 debuff）

### C. 成长呈现方式

15. **科技树**— 固定路径，看得到终点在哪
16. **营地建设**— 视觉化成长，从空地到要塞，有成就感
17. **NPC 好感度**— 和铁匠多交易 → 解锁他的故事 + 更高级物品
18. **收藏柜/博物馆**— 展示在战斗中找到的稀有物品/击杀的特殊敌人
19. **技能卡组**— 像 Slay the Spire 那样选择要带入下一关的"卡牌"（策略组合）
20. **装备系统**— 给指挥官装备不同武器/防具，改变战斗能力

### D. 跨局内容

21. **难度递增+奖励递增**— 通关后解锁更高难度，给更多货币
22. **每日挑战**— 每天一个特殊关卡配置，完成给额外奖励
23. **局外任务板**— "下次通关时建造 3 座箭塔" → 完成给奖励
24. **章节解锁**— 4 关只是第一章，通关解锁第二章新地图
25. **遗物系统**— 稀有的一次性强力道具，每局只能带一个

### E. 叙事与情感

26. **NPC 对话**— 营地里的角色会随着进度变化对话
27. **事件选择**— 通关后弹出"你救了村民，他们加入你的队伍" vs "你掠夺了资源"
28. **分支剧情**— 选择影响后面的关卡内容和敌人
29. **解锁图鉴**— 单位图鉴、敌人图鉴、世界观碎片
30. **指挥官背景**— 随进度解锁指挥官的过去故事

## 三、不同设计下的玩家体验分析

### 设计 1：纯科技树（固定路径升级）

**玩家脑内独白**：
> "我要先点哪条线？先把军团的 HP 点了，打起来肉一点。诶但是经济线也很香啊……算了先走经济，金币多了布局更灵活。"

**行为变化**：每局结束后急着去看科技树，算自己还差多少星辰石。会刻意重打旧关来攒货币。
**风险**：变成了"打工"——玩家不是为了好玩而打，是为了攒钱。如果数值只是 +5% HP，感觉不到变化，会觉得在刷。
**适合**：喜欢规划、喜欢"养成"的玩家。

---

### 设计 2：营地建设（视觉成长）

**玩家脑内独白**：
> "哇我的营地从一块空地变成有小兵营和箭塔了！下次要攒够资源盖个法师塔。看这个营地越来越像回事了。"

**行为变化**：玩家会有"这是我的基地"的归属感。会截图给朋友看"看我建的营地"。即使数值提升和科技树一样，但**看得见**的成长让满足感翻倍。
**风险**：前期营地空荡荡，反而有挫败感。需要设计好"空营地"也不难看的初始状态。
**核心差异**：和科技树本质相同，但**情感回报完全不同**。科技树是一个菜单，营地是你的家。

---

### 设计 3：战斗中拾荒（轻度）

**玩家脑内独白**：
> "这边角落有个物资箱！去捡一下。这个精英怪死了掉了个发光的东西，是稀有材料吗？这局收获不错，带回去了。"

**行为变化**：玩家打一局时**不再只想着赢**，还会想"顺便多探索一下看看有没有好东西"。地图的角落不再是没用的背景，变成了值得去看的地方。输了也不完全沮丧——"虽然输了但捡到了一个稀有材料"。
**核心差异**：把一局的目标从二元（赢/输）变成了多维（赢了+捡了3个材料 / 输了但捡到了稀有货）。**降低了失败的心理惩罚**。

---

### 设计 4：多商贩/NPC 商店

**玩家脑内独白**：
> "铁匠这次卖的东西一般，去军需官看看……哦他有新货了，一个建筑降费的升级，但我星辰石不够。黑市这次刷了个稀有道具，只存在一局，买不买呢？"

**行为变化**：玩家在局外不只是"点科技树"，而是**逛街**。黑市的随机刷新让每次回来都有新鲜感。会犹豫"钱要存着等好东西还是现在就花"。
**风险**：随机性可能让玩家觉得不公平（为什么别人刷到了好的我没有）。黑市如果不限制可能变成赌博感。
**核心差异**：给局外时间增加了**决策的乐趣**，不只是点按钮，而是取舍和判断。

---

### 设计 5：消耗品/道具带入战斗

**玩家脑内独白**：
> "这关听说很难，我带一个增援令进去吧，关键时刻能多召一波兵。再加一个治疗卷轴，保底用。等等，道具栏就两个格子，带哪个？"

**行为变化**：战斗开始前多了一层策略——带什么进去。战斗中多了一个决策——道具什么时候用最划算。**每次战斗的起点不同**，因为带的道具不同。
**核心差异**：让局外的成长直接变成**局内的战术选择**，不是被动加数值，而是给玩家新的工具。

---

### 设计 6：NPC 好感/剧情

**玩家脑内独白**：
> "铁匠上次说他的家乡被毁了，我多帮他升级几次兵营看看他后面说什么。……哇他说完故事后给我打折了！还解锁了一个特殊兵种！"

**行为变化**：玩家升级不只是为了数值，还为了看故事。会刻意选择"帮哪个 NPC"来解锁他们的剧情线。对游戏世界产生情感连接。
**风险**：如果故事写得不好反而尴尬。需要故事质量有保证。
**核心差异**：给数值系统套了一层**情感外衣**，让"花 15 星辰石升级攻击力"变成"帮老铁匠重建锻炉"。

---

### 设计 7：局外任务板（Meta Quest）

**玩家脑内独白**：
> "任务板上写着'下次战斗建造 3 座箭塔'，好，那我这次就多造箭塔。咦还有一个'在 5 分钟内通关'的挑战，要不要试试同时完成？"

**行为变化**：玩家打一局的方式会被任务引导改变。本来的最优解可能是速推，但任务让你尝试别的打法。**增加了重玩时的多样性**。
**风险**：可能让玩家觉得被"作业"支配，从自由游戏变成完成任务清单。
**核心差异**：不改变数值，改变**玩家自己给自己设定的玩法目标**。

## 四、组合对比：玩家体验时间线

### 组合 A：科技树 + 营地建设
```
打完一关 → 看到营地多了个建筑，有成就感 → 点科技树，做规划
→ 下一关 → 打法和上关差不多，只是数值略强 → 循环
```
**玩家感受**：稳步成长，像种田，安定但可能单调。

### 组合 B：拾荒 + 多商贩 + 消耗品
```
打一关 → 边打边探索捡东西 → 结算时看收获 → 去商人那逛逛
→ 买了个消耗品 → 下一关带进去 → 战术变了
```
**玩家感受**：每次回来都有新鲜事，每次出发都带不同东西，探索感+策略感。

### 组合 C：营地建设 + NPC 好感 + 剧情
```
打完一关 → 回营地看到 NPC 新对话 → 帮他升级 → 解锁故事
→ 继续打关 → 阵营变化，敌人也变 → 想看后续剧情
```
**玩家感受**：对世界和角色有感情，想知道后面发生什么，推着你往下走。

### 组合 D：拾荒 + 营地 + 消耗品（轻混合）
```
打一关 → 顺便捡了点材料 → 回营地，用材料建了个新设施
→ 设施解锁了新消耗品 → 带上下关 → 战术选择更多了
```
**玩家感受**：每层系统互相串联，不是独立的菜单。拾荒 → 营地 → 消耗品 → 战斗 → 拾荒，形成闭环。
