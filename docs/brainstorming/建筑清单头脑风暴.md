# 建筑清单头脑风暴 — 60+ 候选（贴图扩展用）

## Context

当前游戏有 4 种建筑（城堡 / 兵营 / 箭塔 / 城墙），加上 archery（射箭场）和 monastery（修道院）共 6 种场景。用户计划用 **可灵 AI + Pixelorama** 流水线批量扩展建筑贴图。本文档是贴图扩展前的**候选建筑清单**，用于：

1. 给后续 AI 生图提供明确的 prompt 主题
2. 给每栋建筑一份**视觉特征描述**（顶视角像素艺术需要哪些辨识度元素）
3. 与已有文档协同：
   - [建筑计划.md](建筑计划.md) — 给**现有 4 种建筑**设计效果（功能层）
   - [建筑技能创意大全.md](建筑技能创意大全.md) — 军功系统 & 士兵成长（系统层）
   - **本文档** — 给**新建筑**列视觉候选（贴图层）

> 已在"建筑计划.md"第五节提过的 10 个新建筑构想，本文会标 `[已有]`，不重复展开，只补充视觉特征。

---

## 使用方式

1. 第一批从下面挑 3-5 栋（推荐组合见文末"第一批推荐"）
2. 每栋按"视觉特征"列在可灵 AI 的 prompt 里描述
3. 命名规范：`{color}_{building_id}/building.png`（沿用现有 [assets/buildings/](../../assets/buildings/) 结构）
4. 贴图规格：320×256、单帧、PNG、5 色变体

## 第一批推荐（3-5 栋试点）

从**视觉辨识度 + 功能互补 + 与现有 6 种建筑差异最大**三个维度挑：

| 推荐 | 建筑 | 理由 |
|---|---|---|
| ⭐⭐⭐ | **市场**（market） | 蓝色布篷 + 摊位结构，和现有石质建筑差异最大；经济类代表 |
| ⭐⭐⭐ | **铁匠铺**（blacksmith） | 熔炉火光 + 暖色调，打破现有冷色主导；科技类代表 |
| ⭐⭐⭐ | **马厩**（stable） | 马匹 + 木质围栏，引入生物元素；军事类代表 |
| ⭐⭐ | **城门**（gate） | 巨大铁栅 + 纵向构图，打破现有横向建筑；防御类代表 |
| ⭐⭐ | **金矿**（mine） | 黑色矿坑 + 暖色岩石，地下元素；经济类补充 |

如果只挑 3 栋：**市场 + 铁匠铺 + 马厩**（覆盖经济/科技/军事 3 类，视觉差异最大）。
如果挑 5 栋：上述全部。

## 贴图规格提醒（生成时参考）

- **分辨率**：320×256（建筑专用）
- **视角**：纯顶视角 top-down，**不要**等距/45°斜视
- **风格关键词**（prompt 模板用）：
  ```
  top-down view, 2D pixel art game asset, a medieval [BUILDING_NAME],
  thick black outline, flat shading, hard pixel edges, no anti-aliasing,
  in the style of the reference images, bright cartoon colors,
  transparent background, 320x256 ratio, Tiny Swords pack style
  ```
- **5 色变体**：blue / red / yellow / black / purple（AI 只出一色，Pixelorama 占位色批量换色）
- **命名**：`assets/buildings/{color}_{building_id}/building.png`

详细工作流见方法论文档（plan 文件 `C:\Users\haoyu\.claude\plans\cuddly-rolling-blanket.md`）。
